&#39;<a href="http://localhost/keyword/%E6%96%9C%E9%87%8C%E5%B2%B3">斜里岳</a>&#39;（しゃりだけ）は北海道の知床半島にある火山。標高は1,547m。<a href="http://localhost/keyword/%E6%96%9C%E9%87%8C%E5%B2%B3">斜里岳</a>道立自然公園に属し、日本百名山に選定されている『日本百名山』深田久弥（著）、朝日新聞社、1<a href="http://localhost/keyword/982%E5%B9%B4">982年</a>、ISBN 4-02-260871-4。清里町の観光スポットの一つである。<br />
<br />
== 概要 ==<br />
古くはアイヌ語でオンネヌプリ（年老いた山）と呼んでいた出典 : [http://www.kiyosatokankou.com/sharidake/ <a href="http://localhost/keyword/%E6%96%9C%E9%87%8C%E5%B2%B3">斜里岳</a>登山ガイド <a href="http://localhost/keyword/%E6%96%9C%E9%87%8C%E5%B2%B3">斜里岳</a>　実話・逸話・裏話] - きよさと観光協会]、20<a href="http://localhost/keyword/14%E5%B9%B4">14年</a>10月閲覧。シャリとはアイヌ語で「葦の生えた湿原」の意。<br />
<br />
山頂部は<a href="http://localhost/keyword/%E6%96%9C%E9%87%8C%E5%B2%B3">斜里岳</a>、南<a href="http://localhost/keyword/%E6%96%9C%E9%87%8C%E5%B2%B3">斜里岳</a>、西峰からなる。オホーツク富士、斜里富士とも呼ばれる。二等三角点名は「<a href="http://localhost/keyword/%E6%96%9C%E9%87%8C%E5%B2%B3">斜里岳</a>」[http://sokuseikagis1.gsi.go.jp/ 国土地理院 基準点成果等閲覧サービス]（国土地理院、20<a href="http://localhost/keyword/13%E5%B9%B4">13年</a>6月28日閲覧）。<br />
<br />
火山の基底は直径約12km出典 : [http://gbank.gsj.jp/volcano-QV/VOL_JP/vol/221.htm 日本の第四紀火山 <a href="http://localhost/keyword/%E5%88%A9%E5%B0%BB%E5%B1%B1">利尻山</a>] - 産業技術総合研究所 地質調査総合センター、20<a href="http://localhost/keyword/14%E5%B9%B4">14年</a>10月閲覧、山体比高950m。活動時期は約30万〜25万年前約28～25万年前という記述もある。出典 : [http://www.pref.hokkaido.lg.jp/ks/skn/environ/parks/syaridake.htm <a href="http://localhost/keyword/%E6%96%9C%E9%87%8C%E5%B2%B3">斜里岳</a>道立自然公園] - 北海道、20<a href="http://localhost/keyword/14%E5%B9%B4">14年</a>10月閲覧。中央火山は溶岩円頂丘。山頂には6個の爆裂火口がある。<br />
<br />
== 登山 ==<br />
山頂には展望がある。登山道は3つあるが、清里町側から登るルートが一般的で、途中で沢筋の旧道と尾根筋の新道に分岐する出典 : [http://www.town.kiyosato.hokkaido.jp/kankou/sharidake/ 観光情報<a href="http://localhost/keyword/%E6%96%9C%E9%87%8C%E5%B2%B3">斜里岳</a>] - 清里町、20<a href="http://localhost/keyword/14%E5%B9%B4">14年</a>10月閲覧。登山口には山小屋「清岳荘」がある。<br />
