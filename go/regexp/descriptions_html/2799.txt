数学の位相空間論周辺分野における&#39;<a href="http://localhost/keyword/%E9%9B%A2%E6%95%A3%E7%A9%BA%E9%96%93">離散空間</a>&#39;（りさんくうかん、）は、その点がすべてある意味で互いに「孤立」しているような空間で、位相空間（またはそれと同様の構造）の非常に単純で極端な例の一つを与える。<br />
<br />
== 定義 ==<br />
&#39;&#39;X&#39;&#39; を集合とする。<br />
* &#39;&#39;X&#39;&#39; 上の&#39;離散位相&#39;  とは、&#39;&#39;X&#39;&#39; の任意の部分集合を &#39;&#39;X&#39;&#39; の開集合とすることによって定まる位相をいう（このとき全ての部分集合は閉集合にもなる）。&#39;&#39;X&#39;&#39; が&#39;離散位相空間&#39;であるとは、それが離散位相を備えた位相空間であることをいう。<br />
* &#39;&#39;X&#39;&#39; 上の&#39;離散一様系&#39; とは、&#39;&#39;X&#39;&#39; の対角集合 &amp;Delta; = {(&#39;&#39;x&#39;&#39;, &#39;&#39;x&#39;&#39;) : &#39;&#39;x&#39;&#39; &amp;isin; &#39;&#39;X&#39;&#39;} &amp;sub; &#39;&#39;X&#39;&#39; × &#39;&#39;X&#39;&#39; の任意の部分集合を近縁と定めることによって与えられる一様系をいう。&#39;&#39;X&#39;&#39; が&#39;離散一様空間&#39;であるとは、それが離散一様系を備えた一様空間であることをいう。<br />
* &#39;&#39;X&#39;&#39; 上の&#39;離散距離&#39; &amp;rho; は<br />
 \\rho(x,y) = \\begin{cases}<br />
   1 &amp;\\mbox{if}\\ x\<br />
eq y , \\\\<br />
   0 &amp;\\mbox{if}\\ x = y<br />
\\end{cases}\\quad(x,y \\in X)<br />
で与えられる。このとき、距離空間 (&#39;&#39;X&#39;&#39;, &amp;rho;) は&#39;離散距離空間&#39;あるいは&#39;孤立点集合&#39;と呼ばれる。<br />
* 距離空間 (&#39;&#39;X&#39;&#39;, &#39;&#39;d&#39;&#39;) の部分集合 &#39;&#39;S&#39;&#39; が &#39;&#39;X&#39;&#39; において&#39;離散&#39;であるとは、&#39;&#39;S&#39;&#39; の各点 &#39;&#39;x&#39;&#39; に対し、適当な &amp;delta; &amp;gt; 0 が（&#39;&#39;x&#39;&#39; ごとに）存在して、&#39;&#39;x&#39;&#39; 以外の &#39;&#39;S&#39;&#39; の各点 &#39;&#39;y&#39;&#39; に対して &#39;&#39;d&#39;&#39;(&#39;&#39;x&#39;&#39;, &#39;&#39;y&#39;&#39;) &amp;gt; &amp;delta; とできるときにいう。このような集合は孤立点から成る。また、部分集合 &#39;&#39;S&#39;&#39; が距離空間 &#39;&#39;X&#39;&#39; において&#39;一様離散&#39;であるとは、適当な定数 &amp;epsilon; &amp;gt; 0 が存在して、&#39;&#39;S&#39;&#39; の任意の相異なる二点に対して &#39;&#39;d&#39;&#39;(&#39;&#39;x&#39;&#39;, &#39;&#39;y&#39;&#39;) &amp;gt; &amp;epsilon; とできるときにいう。<br />
<br />
距離空間 (&#39;&#39;E&#39;&#39;, &#39;&#39;d&#39;&#39;) が&#39;一様<a href="http://localhost/keyword/%E9%9B%A2%E6%95%A3%E7%A9%BA%E9%96%93">離散空間</a>&#39;であるとは、適当な定数 &#39;&#39;r&#39;&#39; &amp;gt; 0 が存在して、&#39;&#39;E&#39;&#39; の任意の二点 &#39;&#39;x&#39;&#39;, &#39;&#39;y&#39;&#39; について、&#39;&#39;x&#39;&#39; = &#39;&#39;y&#39;&#39; か &#39;&#39;d&#39;&#39;(&#39;&#39;x&#39;&#39;, &#39;&#39;y&#39;&#39;) &amp;gt; &#39;&#39;r&#39;&#39; のいづれかが成立することをいう。このとき、台となる距離空間の位相は、距離空間が一様離散であるにもかかわらず、離散位相になる。例えば、実数全体の成す集合の通常の距離に関して、集合 {1, 1/2, 1/4, 1/8, …} がそのような空間の例を与える。<br />
<br />
== 性質 ==<br />
離散距離空間上の一様系は離散一様系であり、離散一様空間上の位相は離散位相である。故に、先に<a href="http://localhost/keyword/%E9%9B%A2%E6%95%A3%E7%A9%BA%E9%96%93">離散空間</a>として挙げたいくつかの概念は、互いに両立する。他方、一様空間あるいは距離空間として離散でないものの中に、その位相が離散位相となるものが存在する。例えば、実数直線における通常の距離からくる距離空間 &#39;&#39;X&#39;&#39;&amp;nbsp;:= {1/&#39;&#39;n&#39;&#39;&amp;nbsp;: &#39;&#39;n&#39;&#39;&amp;nbsp;= 1, 2, 3, …} を考えると、これが離散距離空間でないこと、また（完備でないから）一様空間としても離散でないことは明らかである。にもかかわらず、これは離散位相を備えた離散位相空間になる。すなわち、この &#39;&#39;X&#39;&#39; は「位相的に離散」だが、「一様離散」でも「距離的に離散」でもないということになる。<br />
<br />
さらに以下のようなことが成り立つ。<br />
* <a href="http://localhost/keyword/%E9%9B%A2%E6%95%A3%E7%A9%BA%E9%96%93">離散空間</a>の位相次元は 0 である。<br />
* 位相空間が離散であるための必要十分条件は、その一元集合が必ず開になることであり、あるいはそれが集積点を一切含まないことである。<br />
* 単元集合の全体は、離散位相に対する開基を成す。<br />
* 一様空間 &#39;&#39;X&#39;&#39; が離散であるための必要十分条件は、その対角集合 &amp;Delta; が近縁系をなすことである。<br />
* 任意の離散位相空間は各種の分離公理を全て満たす。特に、任意の<a href="http://localhost/keyword/%E9%9B%A2%E6%95%A3%E7%A9%BA%E9%96%93">離散空間</a>はハウスドルフ空間、つまり分離空間である。<br />
* <a href="http://localhost/keyword/%E9%9B%A2%E6%95%A3%E7%A9%BA%E9%96%93">離散空間</a>がコンパクトであることと、それが有限集合であることとは同値である。<br />
* 任意の離散一様空間あるいは離散距離空間は必ず完備空間である。<br />
* 上二つの事実をあわせれば、任意の離散一様または距離空間が全有界であるための必要十分条件は、それが有限集合であることである。<br />
* 任意の離散距離空間は有界空間である。<br />
* 任意の<a href="http://localhost/keyword/%E9%9B%A2%E6%95%A3%E7%A9%BA%E9%96%93">離散空間</a>は第一可算空間であり、さらに第二可算空間であることと可算であることとが同値になる。<br />
* 少なくとも二点を含む任意の<a href="http://localhost/keyword/%E9%9B%A2%E6%95%A3%E7%A9%BA%E9%96%93">離散空間</a>は完全不連結である。<br />
* 任意の空でない<a href="http://localhost/keyword/%E9%9B%A2%E6%95%A3%E7%A9%BA%E9%96%93">離散空間</a>は、ベールの第二類である。<br />
* 濃度が同じ二つの<a href="http://localhost/keyword/%E9%9B%A2%E6%95%A3%E7%A9%BA%E9%96%93">離散空間</a>は互いに同相である。<br />
* 任意の<a href="http://localhost/keyword/%E9%9B%A2%E6%95%A3%E7%A9%BA%E9%96%93">離散空間</a>は離散距離によって距離化可能である。<br />
* 有限空間が距離化可能なのは、それが<a href="http://localhost/keyword/%E9%9B%A2%E6%95%A3%E7%A9%BA%E9%96%93">離散空間</a>であるときに限る。<br />
* &#39;&#39;X&#39;&#39; が位相空間で &#39;&#39;Y&#39;&#39; が離散位相を備えた集合ならば、&#39;&#39;X&#39;&#39; は &#39;&#39;X&#39;&#39; &amp;times; &#39;&#39;Y&#39;&#39; 二よって十分に被覆される（射影が所期の被覆になる）。<br />
<br />
離散位相空間から他の位相空間への任意の写像は連続であり、離散一様空間から他の一様空間への任意の写像は一様連続になる。つまり、<a href="http://localhost/keyword/%E9%9B%A2%E6%95%A3%E7%A9%BA%E9%96%93">離散空間</a> &#39;&#39;X&#39;&#39; は位相空間と連続写像の圏および一様空間と一様連続写像の圏における &#39;&#39;X&#39;&#39; 上の自由対象である。これらのことは、離散構造が集合上自由であるというより広い現象の例になっている。<br />
<br />
距離空間の場合は、距離空間の圏においては射の取りようによって複数の圏を考えうるから、事態はより複雑になる。射として一様連続写像の全体や連続写像の全体を取れば、確かに離散位相空間は自由だが、これでは一様構造や位相構造について考えただけで、距離構造については何も言っていないに等しい。距離構造についてより関連のある圏は、射をリプシッツ連続写像や弱縮小写像に限ればよいが、これらの圏は（二元以上を持つ集合上で）自由対象を持たない。それでも、離散距離空間は有界距離空間とリプシッツ連続写像の圏における自由対象であり、1 で押さえられる有界距離空間と弱縮小写像の圏における自由対象となる。すなわち、離散距離空間からベルの有界距離空間への任意の写像はリプシッツ連続になり、離散距離空間から別の 1 で押さえられる有界距離空間への任意の写像は弱縮小になる。<br />
<br />
別な方向で考えると、位相空間 &#39;&#39;Y&#39;&#39; から<a href="http://localhost/keyword/%E9%9B%A2%E6%95%A3%E7%A9%BA%E9%96%93">離散空間</a> &#39;&#39;X&#39;&#39; への写像 &#39;&#39;f&#39;&#39; が連続になるための必要十分条件は、それが局所定数函数になる（つまり、&#39;&#39;Y&#39;&#39; の各点の近傍でその上で &#39;&#39;f&#39;&#39; が定数となるようなものが存在する）になることである。<br />
<br />
== 応用例 ==<br />
離散構造は、集合上にほかに自然な位相や一様系、距離が入らないときの「何もしない構造」としてもよく用いられる。また、離散構造は特定の仮定における「極端な」例としても用いられる。例えば、任意の群は離散位相を与えることにより位相群と考えることができ、それにより位相群に対する結果を任意の群に対して適用することができる。実際、代数学で研究されてきた通常の非位相群について、解析学的に離散群として言及することがある。これはいくつかの場合において実際に有効に応用されており、例えば、ポントリャーギン双対などが得られている。0-次元位相多様体（あるいは可微分多様体や解析的多様体）は離散位相空間に他ならないから、任意の離散群を 0-次元リー群と見ることもできる。<br />
<br />
自然数全体の成す<a href="http://localhost/keyword/%E9%9B%A2%E6%95%A3%E7%A9%BA%E9%96%93">離散空間</a>の可算無限個のコピーの直積は、無理数全体の成す空間に同相であり、同相写像は連分数展開によって与えられる。二点から成る<a href="http://localhost/keyword/%E9%9B%A2%E6%95%A3%E7%A9%BA%E9%96%93">離散空間</a> &amp;#x7b;0, 1&amp;#x7d; の可算無限個のコピーの直積はカントール集合に同相であり、この直積に直積一様系を考えれば、実は一様同相になる。この同相写像は三進展開から得られる（カントール空間を参照）。<br />
<br />
数学基礎論において、&amp;#x7b;0, 1&amp;#x7d; の積のコンパクト性の研究は、（<a href="http://localhost/keyword/%E9%81%B8%E6%8A%9E%E5%85%AC%E7%90%86">選択公理</a>よりも弱い）超フィルター原理への位相的取り組みにおいて中心的である。<br />
<br />
== 密着位相 ==<br />
<br />
<a href="http://localhost/keyword/%E9%9B%A2%E6%95%A3%E7%A9%BA%E9%96%93">離散空間</a>の対極にあるのが密着空間である（密着空間の位相は自明位相とも呼ばれる）。これは開集合の数が可能な限り最小（つまり空集合と全体集合のみ）となるような空間である。離散位相が始対象・自由対象であるのに対して、密着位相は終対象・余自由対象になる。つまり、位相空間「から」密着空間「への」任意の写像は連続になる、などの性質がなりたつ。<br />
