下着用の<a href="http://localhost/keyword/%E3%82%AD%E3%83%A3%E3%83%9F%E3%82%BD%E3%83%BC%E3%83%AB">キャミソール</a> <br />
<a href="http://localhost/keyword/%E3%82%AD%E3%83%A3%E3%83%9F%E3%82%BD%E3%83%BC%E3%83%AB">キャミソール</a>を着た女性<br />
<a href="http://localhost/keyword/%E3%82%AD%E3%83%A3%E3%83%9F%E3%82%BD%E3%83%BC%E3%83%AB">キャミソール</a>を横から見た様子<br />
&#39;<a href="http://localhost/keyword/%E3%82%AD%E3%83%A3%E3%83%9F%E3%82%BD%E3%83%BC%E3%83%AB">キャミソール</a>&#39;（仏語：&#39;Camisole&#39;）は、細い肩紐で吊るし肩を露出する形状の袖なしの女性用の上半身用下着ないし上衣である。トップラインは、バストの上でほぼ水平な直線的である（<a href="http://localhost/keyword/%E3%82%AD%E3%83%A3%E3%83%9F%E3%82%BD%E3%83%BC%E3%83%AB">キャミソール</a>トップライン）ことを特徴とするが、前側は、ブラジャーと同様な二つのゆるやかな山形ラインをもつものもある。また、背側が大きく開いたベアバックタイプのスリップで、首の後ろで紐を結んで吊るすホルターネックのものも<a href="http://localhost/keyword/%E3%82%AD%E3%83%A3%E3%83%9F%E3%82%BD%E3%83%BC%E3%83%AB">キャミソール</a>と呼ぶ場合がある。<br />
<br />
下着としては、丈はウェストから腰丈までの短めのものが多い。上着としては、<a href="http://localhost/keyword/%E3%82%AD%E3%83%A3%E3%83%9F%E3%82%BD%E3%83%BC%E3%83%AB">キャミソール</a>ドレス、ないしキャミドレとよび、<a href="http://localhost/keyword/%E3%82%AD%E3%83%A3%E3%83%9F%E3%82%BD%E3%83%BC%E3%83%AB">キャミソール</a>トップラインをもつドレスのことで、丈は下着の場合と異なり、通常のワンピース丈のものまでさまざまである。<br />
<br />
== インナーとしての<a href="http://localhost/keyword/%E3%82%AD%E3%83%A3%E3%83%9F%E3%82%BD%E3%83%BC%E3%83%AB">キャミソール</a> ==<br />
インナーウェアとしては、ランジェリーまたは肌着（アウター調インナー）に分類される。身生地としては、ランジェリーの場合は、フィット感や肌触りの優れたポリエステル、ナイロン、キュプラなどのトリコットが多いが、セクシーないし繊細なテーストでは、ジョーゼット、<a href="http://localhost/keyword/%E3%83%81%E3%83%A5%E3%83%BC%E3%83%AB">チュール</a>ネット、サテンなど、「透け感」や光沢感のある素材が用いられる。また、ファンデーションのような補整力を与える場合は、パワーネットも部分的に利用される。肌着（アウター調インナー）としては、保温性や吸汗性の高いストレッチ天竺やフライス、リブ、スムースなどが使用される。<br />
<br />
=== <a href="http://localhost/keyword/%E3%82%AD%E3%83%A3%E3%83%9F%E3%82%BD%E3%83%BC%E3%83%AB">キャミソール</a>下着のバリエーション ===<br />
; ロングキャミ<br />
: 丈が長めの<a href="http://localhost/keyword/%E3%82%AD%E3%83%A3%E3%83%9F%E3%82%BD%E3%83%BC%E3%83%AB">キャミソール</a>。スリップ。<br />
; ダブルストラップ<br />
: 二組のストラップをもった<a href="http://localhost/keyword/%E3%82%AD%E3%83%A3%E3%83%9F%E3%82%BD%E3%83%BC%E3%83%AB">キャミソール</a>。装飾的。<br />
; ひも結び<br />
: 前後のストラップが分かれていて、肩口や前で結ぶタイプ。<br />
; ホルターネック<br />
: 前の左右のストラップを首の後ろで結ぶタイプ。<br />
; ブラキャミ<br />
: ブラジャーの機能を持たせている<a href="http://localhost/keyword/%E3%82%AD%E3%83%A3%E3%83%9F%E3%82%BD%E3%83%BC%E3%83%AB">キャミソール</a>。胸部に<a href="http://localhost/keyword/%E3%82%AB%E3%83%83%E3%83%97">カップ</a>・パッド・不織布などが付いている。思春期で乳房が成長中（途中で初経を挟む約<a href="http://localhost/keyword/4%E5%B9%B4">4年</a>間[http://www.wacoal.jp/top/tanoshimu_contents/tanken/junior/growth/index.html ワコール探検隊｜「少女」から「おとな」へ約<a href="http://localhost/keyword/4%E5%B9%B4">4年</a>間で変化する成長期のバスト]）に着用するジュニアブラジャーの機能を持たせているブラキャミもある。<br />
<br />
=== <a href="http://localhost/keyword/%E3%82%AD%E3%83%A3%E3%83%9F%E3%82%BD%E3%83%BC%E3%83%AB">キャミソール</a>下着の定義の別説 ===<br />
下着の<a href="http://localhost/keyword/%E3%82%AD%E3%83%A3%E3%83%9F%E3%82%BD%E3%83%BC%E3%83%AB">キャミソール</a>の定義の別説として、丈が短めのミニスリップ全般の意味で<a href="http://localhost/keyword/%E3%82%AD%E3%83%A3%E3%83%9F%E3%82%BD%E3%83%BC%E3%83%AB">キャミソール</a>と呼ぶことがある。この場合、細い肩紐を持たず、身ごろをそのまま上に伸ばしたラウンドタイプのミニスリップも<a href="http://localhost/keyword/%E3%82%AD%E3%83%A3%E3%83%9F%E3%82%BD%E3%83%BC%E3%83%AB">キャミソール</a>に含まれる。他方、<a href="http://localhost/keyword/%E3%82%AD%E3%83%A3%E3%83%9F%E3%82%BD%E3%83%BC%E3%83%AB">キャミソール</a>トップラインをもっていても丈の長めのロング<a href="http://localhost/keyword/%E3%82%AD%E3%83%A3%E3%83%9F%E3%82%BD%E3%83%BC%E3%83%AB">キャミソール</a>は、スリップに分類される。商品をキーワード検索する際には注意が必要である。<br />
<br />
== アウターとしての<a href="http://localhost/keyword/%E3%82%AD%E3%83%A3%E3%83%9F%E3%82%BD%E3%83%BC%E3%83%AB">キャミソール</a> ==<br />
アウターウェアとしては、ドレス、またはカットソーに分類される。<a href="http://localhost/keyword/%E3%82%AD%E3%83%A3%E3%83%9F%E3%82%BD%E3%83%BC%E3%83%AB">キャミソール</a>ドレスは、フォーマルなドレスとして、シャンタン、<a href="http://localhost/keyword/%E3%82%AA%E3%83%BC%E3%82%AC">オーガ</a>ンジー、サテンなどが利用される。カットソーのものは、カジュアルな衣服として、天竺を中心に、シャーリング、スラブ、ストレッチなどのア<a href="http://localhost/keyword/%E3%83%AC%E3%83%B3%E3%82%B8">レンジ</a>が加えられる。<br />
<br />
== 上着と下着の境界の不明瞭化 ==<br />
元来は、日本では、TPOの限られたフォーマルドレスを除いて、下着の意味に用いられてきたが、下着のテーストをもつ下着風ルックを街着として使うファッションの流行から急速に上衣としての用途が広まった。同じ仕立ての<a href="http://localhost/keyword/%E3%82%AD%E3%83%A3%E3%83%9F%E3%82%BD%E3%83%BC%E3%83%AB">キャミソール</a>を重ね着するなど、上着と下着の境界そのものが不明瞭になりつつあり、アウター調インナー（ニューインナー）として分類されるようになっている。<br />
<br />
== 透けブラ防止のための役割 ==<br />
<a href="http://localhost/keyword/%E3%82%AD%E3%83%A3%E3%83%9F%E3%82%BD%E3%83%BC%E3%83%AB">キャミソール</a>の目的の一つとして透けブラ（ブラウスなどインナートップスからブラジャーが透けてしまうこと）防止のための役割がある。<br />
従前より既婚者を中心にシュミーズがその役割を果たしてきたが、若い女性はシュミーズの垢抜けないイメージを敬遠し、透けブラを承知で直接ブラジャーを着用していたが、<a href="http://localhost/keyword/%E3%82%AD%E3%83%A3%E3%83%9F%E3%82%BD%E3%83%BC%E3%83%AB">キャミソール</a>の普及に伴い、透けブラを隠すために用いられるようになったといえる。<br />
普及当初は、中・高校生は学則によりカラーの<a href="http://localhost/keyword/%E3%82%AD%E3%83%A3%E3%83%9F%E3%82%BD%E3%83%BC%E3%83%AB">キャミソール</a>が禁止されているケースが多かったが、現在では有名無実化し、様々なカラーの<a href="http://localhost/keyword/%E3%82%AD%E3%83%A3%E3%83%9F%E3%82%BD%E3%83%BC%E3%83%AB">キャミソール</a>が着用されている。<br />
