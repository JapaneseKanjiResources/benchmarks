水源かん養<a href="http://localhost/keyword/%E4%BF%9D%E5%AE%89%E6%9E%97">保安林</a>を示す標識（岐阜県下呂市）<br />
&#39;<a href="http://localhost/keyword/%E4%BF%9D%E5%AE%89%E6%9E%97">保安林</a>&#39;（ほあんりん）は、公益目的を達成するために、伐採や開発に制限を加える森林のことである。農林水産大臣または都道府県知事が森林法第25条に基づき<a href="http://localhost/keyword/%E4%BF%9D%E5%AE%89%E6%9E%97">保安林</a>として指定する。この場合、森林とは木竹の生育に供される土地を指し、現時点で生育しているか否かは問われない。目的に合わせて17種の<a href="http://localhost/keyword/%E4%BF%9D%E5%AE%89%E6%9E%97">保安林</a>がある[http://www.rinya.maff.go.jp/j/tisan/tisan/con_2.html <a href="http://localhost/keyword/%E4%BF%9D%E5%AE%89%E6%9E%97">保安林</a>制度,林野庁][http://www.pref.aomori.lg.jp/sangyo/agri/hoanrin.html <a href="http://localhost/keyword/%E4%BF%9D%E5%AE%89%E6%9E%97">保安林</a>,青森県庁]。<br />

== <a href="http://localhost/keyword/%E4%BF%9D%E5%AE%89%E6%9E%97">保安林</a>の面積 ==<br />
20<a href="http://localhost/keyword/11%E5%B9%B4">11年</a>3月現在、重複指定を排除した実面積で1,202万ha。これは森林面積の約47%、国土面積でも約31%に相当する[http://www.rinya.maff.go.jp/j/tisan/tisan/con_2_2_1.html <a href="http://localhost/keyword/%E4%BF%9D%E5%AE%89%E6%9E%97">保安林</a>の種類別面積（延べ面積）,林野庁]。指定される<a href="http://localhost/keyword/%E4%BF%9D%E5%AE%89%E6%9E%97">保安林</a>の多くは、水源かん養<a href="http://localhost/keyword/%E4%BF%9D%E5%AE%89%E6%9E%97">保安林</a>、土砂流出防備<a href="http://localhost/keyword/%E4%BF%9D%E5%AE%89%E6%9E%97">保安林</a>、土砂崩壊防備<a href="http://localhost/keyword/%E4%BF%9D%E5%AE%89%E6%9E%97">保安林</a>であり、<a href="http://localhost/keyword/%E4%BF%9D%E5%AE%89%E6%9E%97">保安林</a>全体の90%以上を占める。<br />
<br />
== <a href="http://localhost/keyword/%E4%BF%9D%E5%AE%89%E6%9E%97">保安林</a>の解除 ==<br />
<a href="http://localhost/keyword/%E4%BF%9D%E5%AE%89%E6%9E%97">保安林</a>の指定目的が消滅したとき（例：保全対象の集落、農地が消滅するなど）、公益上の理由（例：公共用道路の建設、送電施設の設置など）が生じたときに限り解除される。この際、必要に応じて代替施設の設置などを求められることがある。民間企業が営利目的で解除を行うことは事実上不可能である。<br />
<br />
農林水産大臣は、<a href="http://localhost/keyword/%E4%BF%9D%E5%AE%89%E6%9E%97">保安林</a>の指定又は解除をしようとするときは、あらかじめその<a href="http://localhost/keyword/%E4%BF%9D%E5%AE%89%E6%9E%97">保安林</a>予定森林都道府県知事に通知しなければならない(法29条)。<br />
<br />
== <a href="http://localhost/keyword/%E4%BF%9D%E5%AE%89%E6%9E%97">保安林</a>の制限 ==<br />
立木の伐採に関しては都道府県知事への届出（一部については許可）が必要となるほか、家畜の放牧、下草・落葉・土石・樹根の採取、土地の形質の変更（掘削、盛土等）については都道府県知事の許可が必要である。<br />
立木の伐採の強度や伐採後の植栽の方法等に関しては、<a href="http://localhost/keyword/%E4%BF%9D%E5%AE%89%E6%9E%97">保安林</a>に指定される際、森林毎に要件が定められる。<br />
<br />
== 指定目的達成の手段 ==<br />
水源かん養<a href="http://localhost/keyword/%E4%BF%9D%E5%AE%89%E6%9E%97">保安林</a>～防火<a href="http://localhost/keyword/%E4%BF%9D%E5%AE%89%E6%9E%97">保安林</a>については、指定の目的を達成するために必要に応じて国、<a href="http://localhost/keyword/%E5%9C%B0%E6%96%B9%E8%87%AA%E6%B2%BB">地方自治</a>体は治山事業を実施することができる。<br />
<br />
== <a href="http://localhost/keyword/%E4%BF%9D%E5%AE%89%E6%9E%97">保安林</a>の思想 ==<br />
燃料や水資源の供給源として、また、防災機能などを有する森林の喪失は、文明の喪失にも繋がることから古くから森林の伐採を制限する法規制が行われてきた。日本では<a href="http://localhost/keyword/676%E5%B9%B4">676年</a>に、飛鳥川上流の南淵山（現在の奈良県高取町高取山）周辺の森林伐採を禁止した例[http://www.maff.go.jp/hakusyo/rin/h13/html/SB1.2.3.htm 平成<a href="http://localhost/keyword/13%E5%B9%B4">13年</a>林業白書]が日本書紀の記述から読み取れる。江戸時代には、岡山藩の熊沢蕃山が治水のために<a href="http://localhost/keyword/%E4%BF%9D%E5%AE%89%E6%9E%97">保安林</a>的思想を打ち出したほか、江戸幕府も1<a href="http://localhost/keyword/666%E5%B9%B4">666年</a>に諸国山川掟を発し、森林の乱開発を戒め植林を促している。<br />
