description<br />
&#39;<a href="http://localhost/keyword/%E7%B5%90%E6%99%B6%E5%A0%B4%E7%90%86%E8%AB%96">結晶場理論</a>&#39;（けっしょうばりろん）とは、金属イオンの p 軌道、 d 軌道、 f 軌道などのエネルギー準位の分裂を、配位子の持つ負電荷が作る静電場によって説明する理論。<br />
<br />
==概要==<br />
結晶中においてあるイオンの位置に他のイオンが作る静電場の総和を&#39;結晶場&#39;という。例えば金属錯体の場合は、配位子の負電荷が中心金属イオンの位置に作る静電場の総和を同様に結晶場と呼ぶ。<br />
<br />
自由イオンにおいて軌道のエネルギーが縮退していたとしても、結晶場がはたらくことで縮退が解けて分裂する。この分裂を&#39;結晶場分裂&#39;といい、分裂した準位を&#39;シュタルク準位&#39;という。例えば金属錯体においては結晶場によってd 軌道の縮退が解けることで間での電子遷移（ d - d 遷移）による吸収スペクトルが観測できる。<br />
この縮退が解ける原因を配位子の持つ負電荷が作る静電場に求めるのが&#39;<a href="http://localhost/keyword/%E7%B5%90%E6%99%B6%E5%A0%B4%E7%90%86%E8%AB%96">結晶場理論</a>&#39;である。<br />
<br />
== 結晶場 ==<br />
結晶場による電子のポテンシャルエネルギーV_{crys}は次のように表される。<br />
:V_{crys} = \\sum_i \\sum_j \\frac{-eQ_j}<br />
ここで&#39;&#39;i&#39;&#39; は電子の番号、&#39;&#39;j&#39;&#39; は周囲の原子やイオンの番号で&#39;&#39;Qj&#39;&#39; はその電荷である。これを摂動とみなして、ハミルトニアンの固有値問題を解くことを考える。そのためにはV_{crys}を球面調和関数Y_{km}(\\theta_i , \\phi_i)を用いて次のように書き換えると便利である。<br />
:V_{crys} = \\sum_i \\sum_{tp} r_j^t A_{tp} D_p^{(t)} <br />
:A_{tp} = \\sqrt{\\frac{4\\pi}{2t+1}} \\sum_j \\frac{-e Q_j}{R_j^{t+1}} Y_{tp}^* (\\theta_i , \\phi_i)<br />
:D_p^{(t)} = \\sum_i r_i^t C_p^{(t)} (\\theta_i , \\phi_i)<br />
= \\sum_i r_i^t \\sqrt{\\frac{4\\pi}{2t+1}} Y_{tp} (\\theta_i , \\phi_i) <br />
この&#39;&#39;Atp&#39;&#39; を&#39;結晶場パラメータ&#39;という。一般に結晶場の<a href="http://localhost/keyword/%E5%AF%BE%E7%A7%B0%E6%80%A7">対称性</a>のために、独立な結晶場パラメータの数は限られる。例えば、点群&#39;&#39;C1&#39;&#39; 、&#39;&#39;Ci&#39;&#39; 、&#39;&#39;Cs&#39;&#39; の場合を除いて&#39;&#39;p = 1&#39;&#39; と&#39;&#39;p = 5&#39;&#39; の成分はゼロになる。<br />
<br />
=== 球テンソル演算子法 ===<br />
一般に多電子系の波動関数は複雑なので、摂動ハミルトニアンの行列要素を求めることは難しい。しかし多電子系がラッセル–サンダーズ結合を満足しているときは、その波動関数を全角運動量&#39;&#39;J&#39;&#39; とその磁気量子数&#39;&#39;M&#39;&#39; で表現することができる。球テンソル演算子法では、行列要素は3j記号や6j記号を用いて表される。簡単な場合については解析的な表現がいろいろな量子力学の本に表として掲載されている。またこれらを数値的に求めるコンピュータプログラムを示しているものもある。<br />
<br />
== 結晶場分裂 ==<br />
=== 八面体対称場によるd軌道の分裂 ===<br />
3d軌道が八面体対称(点群&#39;&#39;Oh&#39;&#39;)の結晶場中にある場合の分裂を考える。結晶場の<a href="http://localhost/keyword/%E5%AF%BE%E7%A7%B0%E6%80%A7">対称性</a>によってどのような分裂が起こるのかを、シュレーディンガー方程式を解かないで予測するには、点群の既約表現を用いるのが便利である。一般に波動関数は\\Psi_{n,l,m}(r,\\theta,\\phi)=R_{n,l}(r)P_{lm}(\\cos\\theta)\\Phi_m(\\phi)と変数分離でき、3d軌道の場合は次のような形を取る。<br />
:\\Psi_{3,2,m}(r,\\theta,\\phi)=R_{3,2}(r)P_{2m}(\\cos\\theta)\\frac{e^{im\\phi}}{\\sqrt{2}\\pi}<br />
まず点群&#39;&#39;Oh&#39;&#39;の部分群である点群&#39;&#39;O&#39;&#39;について調べる。点群&#39;&#39;O&#39;&#39;の対称操作は回転ばかりだが、変化するのは\\phiについての関数だけなので\\Phi_{m}(\\phi)だけ考えれば良い。&#39;&#39;m&#39;&#39;の5つの値に対応する関数\\Phi_{m}(\\phi)に回転対称操作を行う。<br />
:<br />
\\begin{pmatrix}<br />
e^{2i\\omega} &amp; 0           &amp; 0      &amp; 0            &amp; 0             \\\\<br />
 0           &amp; e^{i\\omega} &amp; 0      &amp; 0            &amp; 0             \\\\<br />
 0           &amp; 0           &amp; e^0    &amp; 0            &amp; 0             \\\\<br />
 0           &amp; 0           &amp; 0      &amp; e^{-i\\omega} &amp; 0             \\\\<br />
 0           &amp; 0           &amp; 0      &amp; 0            &amp; e^{-2i\\omega} \\\\<br />
\\end{pmatrix}<br />
\\begin{pmatrix}<br />
e^{2i\\phi}   \\\\<br />
e^{i\\phi}    \\\\<br />
e^{0}        \\\\<br />
e^{-i\\phi}   \\\\<br />
e^{-2i\\phi}  \\\\<br />
\\end{pmatrix}<br />
=<br />
\\begin{pmatrix}<br />
e^{2i(\\phi+\\omega)}   \\\\<br />
e^{i(\\phi+\\omega)}    \\\\<br />
e^{0}                 \\\\<br />
e^{-i(\\phi+\\omega)}   \\\\<br />
e^{-2i(\\phi+\\omega)}  \\\\<br />
\\end{pmatrix}<br />
<br />
このように５つの3d軌道を基底とする回転操作の表現行列を作ることが出来た。よってこの表現行列の指標(ト<a href="http://localhost/keyword/%E3%83%AC%E3%83%BC%E3%82%B9">レース</a>)は<br />
:\\chi(\\omega)=e^{2i\\omega}+e^{i\\omega}+e^0+e^{-i\\omega}+e^{-2i\\omega}=\\frac{\\sin (5\\omega/2)}{\\sin (\\omega/2)}<br />
となる。また一般に角運動量量子数が&#39;&#39;l&#39;&#39; の軌道のときの指標は次のように表されることが同様の方法からわかる。<br />
:\\chi(\\omega)=\\frac{\\sin [(l+1/2)\\omega]}{\\sin (\\omega/2)}<br />
<br />
これらの軌道を基底とする点群&#39;&#39;O&#39;&#39;のすべての回転操作&#39;&#39;C2&#39;&#39;(ω=π), &#39;&#39;C4&#39;&#39;(ω=π/2), &#39;&#39;C3&#39;&#39;(ω=2π/3)の表現行列の指標は次のように求められる。<br />
<br />
これを簡約公式を用いて既約表現に分解すると：<br />
::点群&#39;&#39;O&#39;&#39; :　&#39;&#39;Γ&#39;&#39;（d軌道) = &#39;&#39;E&#39;&#39; + &#39;&#39;T2&#39;&#39;<br />
したがって、点群&#39;&#39;Oh&#39;&#39;は点群&#39;&#39;O&#39;&#39;に対称心&#39;&#39;i&#39;&#39; を加えてできるものであるから、点群&#39;&#39;O&#39;&#39;に対して得た結果に偶(gerade)か奇(ungerade)かを決めてやれば良い。d軌道はすべてgeradeであるので：<br />
::点群&#39;&#39;Oh&#39;&#39; :　&#39;&#39;Γ&#39;&#39;（d軌道) = &#39;&#39;Eg&#39;&#39; + &#39;&#39;T2g&#39;&#39;<br />
よって５つのd軌道は球対称場の中では縮退しているが、点群&#39;&#39;Oh&#39;&#39;の場の中では縮退が解けて、二重縮退の状態&#39;&#39;Eg&#39;&#39; と三重縮退の状態&#39;&#39;T2g&#39;&#39;に分裂する。<br />

===八面体錯体のd軌道の例===<br />
例えば、正八面体型の6配位の金属錯体について考える。<br />
座標の原点に金属イオンを配置し、 x 軸、 y 軸、 z 軸上に6個の配位子を正八面体型に配置する。<br />
これらの配位子の負電荷が作る結晶場を計算すると各軸上で大きくなる。<br />
そのため、d 軌道のうち軸上に電子密度が大きくなる部分を持つ d_{z2} および、 d_{x2-y2} の2つの軌道は結晶場の影響を他の3つの軌道( d_{xy} 、 d_{yz} 、 d_{zx} )より大きく受ける。<br />
すなわち、これら2つの d 軌道に電子が入ると配位子の負電荷と反発するので、他の3つの軌道に入る場合よりもエネルギーが高いことになる。<br />
このようにして正八面体型の6配位の金属錯体ではエネルギーの高い2つの d 軌道とエネルギーの低い3つのd 軌道に分裂する。<br />
<br />
==弱い結晶場==<br />
スピン軌道相互作用と比べて結晶場が小さい場合を考える。このような弱い結晶場における多電子系の状態では、まず自由原子における状態がLS結合によって分裂し、次にそれらの各状態が結晶場という摂動により分裂する。<br />
<br />
固有状態は項記号&#39;&#39;2S+1Γ&#39;&#39; で表記する。これは原子の項記号と違って、固有状態の既約表現&#39;&#39;Γ&#39;&#39; を用いる。つまり原子の周囲の点対称場から影響を受けるのは軌道に関する固有状態であって、スピンには影響が無い。<br />
<br />
==強い結晶場==<br />
結晶場がスピン軌道相互作用と比べてずっと大きい場合を考える。このような強い結晶場の場合は、LS結合に比べて結晶場の効果が大きい。<br />
<br />
==<a href="http://localhost/keyword/%E7%B5%90%E6%99%B6%E5%A0%B4%E7%90%86%E8%AB%96">結晶場理論</a>の問題点と配位子場理論==<br />
<a href="http://localhost/keyword/%E7%B5%90%E6%99%B6%E5%A0%B4%E7%90%86%E8%AB%96">結晶場理論</a>はd 軌道の分裂の様式を正しく説明することができるが、その分裂の大きさについては説明できない。<br />
<a href="http://localhost/keyword/%E7%B5%90%E6%99%B6%E5%A0%B4%E7%90%86%E8%AB%96">結晶場理論</a>からは同じ価数の陰イオンであれば、同じ分裂の大きさになるという結論になるが実際には分裂の大きさは同じ価数であっても配位子の種類に依存し、I - ＜ Br - ＜ Cl - ＜ F - のようになることが知られている（分光化学系列）。<br />
また、中性の一酸化炭素を配位子とする錯体でd 軌道分裂が大きくなることも説明できない。<br />
分裂の大きさを正しく計算するには分子軌道を考慮した配位子場理論によることが必要である。<br />
