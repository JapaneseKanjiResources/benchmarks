description<br />
<br />
&#39;<a href="http://localhost/keyword/%E7%B2%BE%E5%AF%86%E6%8E%A1%E7%82%B9">精密採点</a>&#39;（せいみつさいてん）とは、第一興商の通信カラオケ機、DAM-G100 (BB cyber DAM) 以降に実装されている採点機能の一種である。音程の正確さだけでなく、しゃくりやビブラートなどの歌唱法を検知し、採点基準に含めている。DAM-G100F1/ - F2 (BB cyber DAM f stage) では「&#39;<a href="http://localhost/keyword/%E7%B2%BE%E5%AF%86%E6%8E%A1%E7%82%B9">精密採点</a>デュエット&#39;」が、DAM-G70 (cyber DAM G70) では「&#39;<a href="http://localhost/keyword/%E7%B2%BE%E5%AF%86%E6%8E%A1%E7%82%B9">精密採点</a>プラス&#39;」（せいみつさいてんプラス）が実装された。また、DAM-XG1000/XG1000II (Premier DAM) とG50X (cyber DAM α) では「<a href="http://localhost/keyword/%E7%B2%BE%E5%AF%86%E6%8E%A1%E7%82%B9">精密採点</a>」に加え、大幅に採点基準が変更された「&#39;<a href="http://localhost/keyword/%E7%B2%BE%E5%AF%86%E6%8E%A1%E7%82%B9">精密採点</a>II&#39;」（せいみつさいてんツー）も実装されており、DAM-XG5000 (LIVE DAM) では<a href="http://localhost/keyword/%E7%B2%BE%E5%AF%86%E6%8E%A1%E7%82%B9">精密採点</a>IIの機能を踏襲した「&#39;<a href="http://localhost/keyword/%E7%B2%BE%E5%AF%86%E6%8E%A1%E7%82%B9">精密採点</a>DX&#39;」（せいみつさいてんデラックス）が実装されている。<br />
<br />
== 概要 ==<br />
<a href="http://localhost/keyword/%E7%B2%BE%E5%AF%86%E6%8E%A1%E7%82%B9">精密採点</a>を利用するには、リモコンあるいは機械本体のキースイッチを用いて、専用のリクエスト番号 8000-01（8000-02・8100-01でも可）を入力する。リモコンの機種によっては、メニュー画面から選択して<a href="http://localhost/keyword/%E7%B2%BE%E5%AF%86%E6%8E%A1%E7%82%B9">精密採点</a>のボタンを押すことでも利用できる（DAMステーションやデンモクiDが該当）。リクエスト番号が受理されると、それ以降の楽曲の演奏中は、画面右上部に採点状況を示すアイコンが表示される。1曲の演奏終了後には、歌唱に対する各観点からの評価と、それらを総合した得点（100点満点）が表示される。終了する際はリクエスト番号8000-01を入力し、演奏中止ボタンを押す。また、演奏終了後一定時間予約がない場合も自動的に終了する。また、機械本体のキースイッチを押した場合標準でcyber DAMは<a href="http://localhost/keyword/%E7%B2%BE%E5%AF%86%E6%8E%A1%E7%82%B9">精密採点</a>、Premier DAMは<a href="http://localhost/keyword/%E7%B2%BE%E5%AF%86%E6%8E%A1%E7%82%B9">精密採点</a>II、LIVE DAMは<a href="http://localhost/keyword/%E7%B2%BE%E5%AF%86%E6%8E%A1%E7%82%B9">精密採点</a>DXが選択される（但し、Premier DAMでも採点オプションで「<a href="http://localhost/keyword/%E7%B2%BE%E5%AF%86%E6%8E%A1%E7%82%B9">精密採点</a>」を選択する事は可能だが、LIVE DAMでは<a href="http://localhost/keyword/%E7%B2%BE%E5%AF%86%E6%8E%A1%E7%82%B9">精密採点</a>DXしか搭載されておらず、<a href="http://localhost/keyword/%E7%B2%BE%E5%AF%86%E6%8E%A1%E7%82%B9">精密採点</a>I・IIを選択する事は出来無い）。<br />
<br />
Iでは音程・しゃくり・ビブラートのアイコンが信号機に見立てた形で右上に表示され、それ等を感知するとアイコンが点灯して点数に勘案されている（音程はピンク、しゃくりは黄色、ビブラートは緑色で表現される）。<br />
<br />
<a href="http://localhost/keyword/%E7%B2%BE%E5%AF%86%E6%8E%A1%E7%82%B9">精密採点</a>プラスは<a href="http://localhost/keyword/%E7%B2%BE%E5%AF%86%E6%8E%A1%E7%82%B9">精密採点</a>のマイナーチェンジ版であり、「リズム」に代わって「こぶし」が登場し、音程は「高音域」「中音域」「低音域」それぞれに対しても評価されるようになった。また<a href="http://localhost/keyword/%E7%B2%BE%E5%AF%86%E6%8E%A1%E7%82%B9">精密採点</a>デュエットは、<a href="http://localhost/keyword/%E7%B2%BE%E5%AF%86%E6%8E%A1%E7%82%B9">精密採点</a>の採点基準を用いて（ビブラートタイプは表示されない）2本のマイクで別々に採点を行い、「ふたりの息ぴったり度」も測定する。<br />
<br />
<a href="http://localhost/keyword/%E7%B2%BE%E5%AF%86%E6%8E%A1%E7%82%B9">精密採点</a>IIでは、「ロング<a href="http://localhost/keyword/%E3%83%88%E3%83%BC%E3%83%B3">トーン</a>（ビブラート）の上手さ」や「フォール」などの新たな採点基準が加わった他、『&#39;見えるガイドメロディー&#39;』という楽譜風のラインにピアノロール若しくは棒グラフ風のバーが画面最上部に表示され、自分の音程によって描かれる『歌唱軌跡』バーと重ね合わせる事で音程の見える化が成されており、歌唱中に「しゃくり」「ビブラート」「フォール」「こぶし」を検知すると、それぞれ対応するアイコンがバー上の検知ポイントに表示される。音程バーはIと同じピンク色で表現されている。また、通常深緑色で表示される点数が、90点以上で銀色、95点以上で金色になり、得点表示の際の演出やBGMもそれによって多少変わる仕様になっている。<br />
<br />
<a href="http://localhost/keyword/%E7%B2%BE%E5%AF%86%E6%8E%A1%E7%82%B9">精密採点</a>DXは<a href="http://localhost/keyword/%E7%B2%BE%E5%AF%86%E6%8E%A1%E7%82%B9">精密採点</a>IIの採点方法を踏襲しているが、曲の最低音と最高音は他とは違う色のバーになっており、音程バーは黄緑色で表現されている他、「しゃくり」「ビブラート」、「こぶし」、「フォール」のアイコンのデザインも異なっている。採点結果は始めの画面で得点が小数第3位まで表示され、大まかな分析がレーダーチャートで表示される他、本人の歌唱に対しての特徴や評価等の<a href="http://localhost/keyword/%E3%82%B3%E3%83%A1%E3%83%B3%E3%83%88">コメント</a>も表示される。次の画面で詳細な採点結果が表示される。79.999点以下、80点、90点、95点以上でBGMが変わり、90点以上の場合は冒頭で歓声や拍手が入ってくる。100点（満点）の場合は、点数の周りに虹色のエフェクトがかかる。<br />
さらに、SMART DAMかつEX<a href="http://localhost/keyword/ILE">ILE</a>の楽曲に限り、EX<a href="http://localhost/keyword/ILE">ILE</a><a href="http://localhost/keyword/%E7%B2%BE%E5%AF%86%E6%8E%A1%E7%82%B9">精密採点</a>DXを利用できる。<a href="http://localhost/keyword/%E7%B2%BE%E5%AF%86%E6%8E%A1%E7%82%B9">精密採点</a>DX-Gは前述の採点方法のマイナーチェンジ版であり、音程バーは通常の楽曲では黄色、デュエット曲はそれぞれのパート別の色（男性は主に青色、女性は赤色）に変更されており、歌唱中は「しゃくり」「ビブラート」「こぶし」「フォール」の回数がリアルタイムで表示され、ビブラートの場合は演奏終了後回数と秒数が併記される。<br />
<br />
表示される点数はIでは50点から100点、II・DXが0点から100点。II以降、音程の細分化やビブラート・ロング<a href="http://localhost/keyword/%E3%83%88%E3%83%BC%E3%83%B3">トーン</a>の良し悪し等も採点に勘案される様になった事による採点基準の増強によって、Iで高得点だった者でもIIやDXでは高得点を出しにくい者が多い。<br />
<br />
I・プラス・II・DXともに、「club DAM MEMBERSHIPカード」もしくは「club DAM MEMBERSアプリ」設定済みのおサイフケータイでログインして採点すると、結果が公式サイト「clubDAM.com」に保存され、後でPCや携帯電話から確認出来る。なお「clubDAM.com」から採点結果を確認すると、DX以外では歌唱後の結果画面で整数でしか表示されなかった得点が、小数点以下3桁まで表示される。また、専用のプリンターが接続されている場合は、有料（300円・Edy決済）で結果を印刷することも可能である。<br />

== 採点基準 ==<br />
<a href="http://localhost/keyword/%E7%B2%BE%E5%AF%86%E6%8E%A1%E7%82%B9">精密採点</a>は、歌唱時に以下のような要素を検知し、演奏終了後に画面で通知する。<br />
; 音程<br />
: 楽曲ごとに設定されているメロディーライン（ガイドメロディー）に対して音程の正確さを判定する。ガイドと声の音程が一致すると、画面右上部の桃色のアイコンが点灯する。また<a href="http://localhost/keyword/%E7%B2%BE%E5%AF%86%E6%8E%A1%E7%82%B9">精密採点</a>II・DXでは、『見えるガイドメロディー』のバーと自分の『歌唱軌跡』のバーが重なるとバーの色が桃色（DXでは黄緑色）となり、音程が合っている事が視覚的に分かる。通知画面では楽曲全体通しての正確さが整数の百分率で表される。<br />
: 加えて、<a href="http://localhost/keyword/%E7%B2%BE%E5%AF%86%E6%8E%A1%E7%82%B9">精密採点</a>プラスでは「高音域」「中音域」「低音域」それぞれの評価が5段階評価で5目盛のメータに示され、<a href="http://localhost/keyword/%E7%B2%BE%E5%AF%86%E6%8E%A1%E7%82%B9">精密採点</a>IIでは「低音」「高音」それぞれの評価が4種類のマーク（◎・○・△・×）で示される。また<a href="http://localhost/keyword/%E7%B2%BE%E5%AF%86%E6%8E%A1%E7%82%B9">精密採点</a>DXでは音程正確率の変化が折れ線グラフで表示される（JOYSOUNDの分析採点にも見られる）。<br />
; しゃくり<br />
: 設定された音程よりも低い音をまず発声し、そこから本来の音程に近づけてゆく。このような歌唱法を機械が検知した場合、しゃくりとして判定される。しゃくりが判定された場合、画面右上部の黄色いアイコンが点灯する（IIではバー上の検知ポイントに配置）。通知画面では合計回数で表される。<br />
; ビブラート<br />
: ある程度の音高を保って声を揺らす歌唱法を指す。ビブラートを機械が検知した場合、採点に加算される。歌唱中にビブラートが検知されると、画面右上部の緑色のアイコンが点灯する（II以降ではバー上の検知ポイントに配置）。通知画面では、1曲中での合計時間（秒単位。1秒未満の場合、小数点以下1桁も表示。I・II・DXは最大99秒迄しか表示されないが、DX－Gは100秒以降の数値もきちんと表示される）と揺れの波形がタイプ別診断で示される（<a href="http://localhost/keyword/%E7%B2%BE%E5%AF%86%E6%8E%A1%E7%82%B9">精密採点</a>デュエットでは波形タイプ診断は省略）。Iではビブラートの多用で点数が上がる事も多いが、II以降だとビブラートを多用するだけでは得点は上がらない事が多い。<br />
; 抑揚<br />
: 抑揚（強弱）について判定し、得点に加算する。楽曲が幾つかのセクターに別れていて、セクターごとに最大音と最小音を比較して評価している。通知画面では、楽曲全体を通しての評価が11段階評価で5目盛のメータに示される。<br />
; 演奏時間<br />
: 結果画面には表示されないが、曲のどこまで歌ったかで得点が変わる。演奏量が規定より少ないと最高点が抑制される。概ね2度めのサビまたは楽曲の後半で得点が漸増し、それ以降が本来の採点点数（100点満点）となる。<br />
; リズム（<a href="http://localhost/keyword/%E7%B2%BE%E5%AF%86%E6%8E%A1%E7%82%B9">精密採点</a>プラスを除く）<br />
: 伴奏に対して歌唱が速い場合には「走り」、遅い場合には「タメ」として判定される。走りは減点対象になっている。これらは7段階のグラフメーターによって表される。<br />
; こぶし（<a href="http://localhost/keyword/%E7%B2%BE%E5%AF%86%E6%8E%A1%E7%82%B9">精密採点</a>プラス、<a href="http://localhost/keyword/%E7%B2%BE%E5%AF%86%E6%8E%A1%E7%82%B9">精密採点</a>II・DX）<br />
: 基本となる旋律の中で音を細かく動かす装飾音的な節回し。こぶしが判定された場合、青いアイコンが画面最上部バー上の検知ポイントに配置される。通知画面では合計回数で表される。<br />
; フォール（<a href="http://localhost/keyword/%E7%B2%BE%E5%AF%86%E6%8E%A1%E7%82%B9">精密採点</a>II・DX）<br />
: &#34;しゃくり&#34;とは逆に、設定された音程よりも高い音をまず発声し、そこから本来の音程に近づける歌唱法。フォールが判定された場合、赤いアイコンが画面最上部バー上の検知ポイントに配置される。通知画面では合計回数で表される。<br />
; ビブラートの上手さ（<a href="http://localhost/keyword/%E7%B2%BE%E5%AF%86%E6%8E%A1%E7%82%B9">精密採点</a>II・DX）<br />
: ビブラートの波形の美しさを評価する。通知画面では、楽曲全体を通しての評価が11段階評価で5個の緑色の★（DXは●）マークで示される。<br />
; ロング<a href="http://localhost/keyword/%E3%83%88%E3%83%BC%E3%83%B3">トーン</a>の上手さ（<a href="http://localhost/keyword/%E7%B2%BE%E5%AF%86%E6%8E%A1%E7%82%B9">精密採点</a>II・DX）<br />
: ロング<a href="http://localhost/keyword/%E3%83%88%E3%83%BC%E3%83%B3">トーン</a>（同一音程で長く音を伸ばす）の美しさを評価する。通知画面では、楽曲全体を通しての評価が11段階評価で5個の桃色の★（DXは●）マークで示される。<br />
; ふたりの息ぴったり度（<a href="http://localhost/keyword/%E7%B2%BE%E5%AF%86%E6%8E%A1%E7%82%B9">精密採点</a>デュエット）<br />
:<br />
; EX<a href="http://localhost/keyword/ILE">ILE</a>指数（EX<a href="http://localhost/keyword/ILE">ILE</a><a href="http://localhost/keyword/%E7%B2%BE%E5%AF%86%E6%8E%A1%E7%82%B9">精密採点</a>DX）<br />
: 80点未満で初ザイル、80点以上83点未満でふつうザイル、83点以上86点未満でアマザイル、86点以上90点未満でウマザイル、90点以上95点未満でプロザイル、95点以上でエグザイル級となる。<br />
