&#39;<a href="http://localhost/keyword/%E7%B5%8C%E7%B1%8D%E5%BF%97">経籍志</a>&#39;（けいせきし）は、中国の紀伝体歴史書における構成要素の一つである「志」の中の篇目名であり、図書目録のことを指して言う。名称は異なるが、『漢書』に始まる「&#39;芸文志&#39;」と内容的には一致するものである。<br />
<br />
== 『隋書』「<a href="http://localhost/keyword/%E7%B5%8C%E7%B1%8D%E5%BF%97">経籍志</a>」 ==<br />
<a href="http://localhost/keyword/%E7%B5%8C%E7%B1%8D%E5%BF%97">経籍志</a>は、『旧唐書』などの後代の正史にも見られるが、その筆頭は、唐の魏徴らによる『隋書』である。『隋書』の「<a href="http://localhost/keyword/%E7%B5%8C%E7%B1%8D%E5%BF%97">経籍志</a>」は、『漢書』「芸文志」以降、魏晋南北朝時代の各人によって<a href="http://localhost/keyword/%E8%A9%A6%E8%A1%8C%E9%8C%AF%E8%AA%A4">試行錯誤</a>が繰り返されてきた図書分類法の集大成として生まれた&#39;四部分類&#39;によって編輯されている。その構成は、<br />
# 経部 <br />
# 史部 <br />
# 子部 <br />
# 集部 <br />
# 道経 <br />
# 仏経 <br />
と、四部の後に道仏二教の経典を付録している。なお、各部は、さらに門類に細分されている。各条には、書名、巻数、撰者を記し、兵乱によって亡失したものには、「亡」と注記している。<br />
<br />
== 『旧唐書』「<a href="http://localhost/keyword/%E7%B5%8C%E7%B1%8D%E5%BF%97">経籍志</a>」 ==<br />
上述の如く、唐の滅亡後、五代後晋の時代に勅命によって編纂された『旧唐書』にも、「<a href="http://localhost/keyword/%E7%B5%8C%E7%B1%8D%E5%BF%97">経籍志</a>」が立てられている。この志も、<br />
# 甲部経録<br />
# 乙部史録<br />
# 丙部子録<br />
# 丁部集録<br />
の四部分類を踏襲している。ただし、『隋書』とは異なり、道仏二教を四部以外に分出してはいない。また、丙部子録中には類書類が新設されている。<br />
<br />
『旧唐書』が撰せられたのは、10世紀初めではあるが、こと「<a href="http://localhost/keyword/%E7%B5%8C%E7%B1%8D%E5%BF%97">経籍志</a>」に関しては、五代の書目ではないので注意が必要である。その収録する典籍の時代は、玄宗の開元年間（<a href="http://localhost/keyword/713%E5%B9%B4">713年</a> - <a href="http://localhost/keyword/741%E5%B9%B4">741年</a>）にまで遡る。玄宗朝の書目である『古今書録』40巻の抄出が「<a href="http://localhost/keyword/%E7%B5%8C%E7%B1%8D%E5%BF%97">経籍志</a>」である。よって、玄宗朝以後の典籍は含まれていないのである。<br />
