コントロールタワー<br />
&#39;<a href="http://localhost/keyword/%E3%83%84%E3%82%A4%E3%83%B3%E3%83%AA%E3%83%B3%E3%82%AF%E3%82%82%E3%81%A6%E3%81%8E">ツインリンクもてぎ</a>&#39;（&#39;Twin Ring Motegi&#39;）は、栃木県芳賀郡茂木町にある自動車<a href="http://localhost/keyword/%E3%83%AC%E3%83%BC%E3%82%B9">レース</a>場である。1<a href="http://localhost/keyword/997%E5%B9%B4">997年</a>8月営業開始。四輪のSUPER GTや、二輪の日本グランプリ（もしくはパシフィックグランプリ）が開催されている。なお、「ツインリン&#39;グ&#39;」（濁音）は誤り。<br />
<br />
== 概要 ==<br />
鈴鹿サーキットを所有するホンダが、日本で2つ目のサーキットとして19<a href="http://localhost/keyword/80%E5%B9%B4">80年</a>代に開発を開始し、1<a href="http://localhost/keyword/997%E5%B9%B4">997年</a>8月に営業が開始された。1<a href="http://localhost/keyword/998%E5%B9%B4">998年</a>には本田技研工業創立50周年記念イベント「ありがとう<a href="http://localhost/keyword/%E3%83%95%E3%82%A7%E3%82%B9">フェス</a>タinもてぎ」が開催された。<br />
<br />
オーバルコースとロードコースの2つのコース（周回路）を併設する、世界でもまれなサーキット施設であり、本田技研工業の100%子会社として作られた。現在は鈴鹿サーキットを運営する鈴鹿サーキットランドと合併した新会社「株式会社モビリティランド」が運営を行っている。<br />
<br />
== コース ==<br />
スーパースピードウェイとロードコース<br />
; スーパースピードウェイ<br />
: 1.5mileのオーバルコース、アメリカンモータースポーツイベント（インディカー・シリーズ・ドラッグ<a href="http://localhost/keyword/%E3%83%AC%E3%83%BC%E3%82%B9">レース</a>など）で主に使用。<br />
: 直線長はフロント、バック共に1,969フィート (600m)。バンク角は全ターン10度。バンク角が浅い上に1〜2ターンに比べると3〜4ターンがきつく、1.5mile級オーバルコースとしては珍しくシフトダウンが必要になる。コースレイアウト自体はマディソン・インターナショナル・<a href="http://localhost/keyword/%E3%83%AC%E3%83%BC%E3%82%B9">レース</a>ウェイを参考にしている。<br />
: 20<a href="http://localhost/keyword/10%E5%B9%B4">10年</a>現在はスーパー耐久の一戦でも使用されるが、使用するマシンにオーバルでのクラッシュ時に対応した衝撃吸収構造などが設けられていない関係から、その際はコース上に仮設シケインを設けている[http://www.twinring.jp/s-taikyu_m/ovalbattle/index.html オーバルバトルとは?] - <a href="http://localhost/keyword/%E3%83%84%E3%82%A4%E3%83%B3%E3%83%AA%E3%83%B3%E3%82%AF%E3%82%82%E3%81%A6%E3%81%8E">ツインリンクもてぎ</a>。<br />
; ロードコース<br />
: 全長4.8kmのテクニカルサーキット。<br />
: ショートカット路面を利用することにより東コース（3.4km）、西コース（1.4km）のショートコースとしても利用可能。<br />
: ほとんどの<a href="http://localhost/keyword/%E3%83%AC%E3%83%BC%E3%82%B9">レース</a>イベントはこちらで行われる。<br />
; 北ショートコース<br />
: 全長982mのショートコース。主にレーシングカートやミニバイクのコースとして使用。<br />
; ダートトラック(20<a href="http://localhost/keyword/12%E5%B9%B4">12年</a>5月末で閉鎖)<br />
: 全長200m、400mのダートオーバルコース。ダートトラック<a href="http://localhost/keyword/%E3%83%AC%E3%83%BC%E3%82%B9">レース</a>のコースとして使用。<br />
; 南コース（マルチコース）<br />
: 20,000平方メートルの多目的コース。モタード<a href="http://localhost/keyword/%E3%83%AC%E3%83%BC%E3%82%B9">レース</a>、ジムカーナ、ライディングスクール他イベントなどで使用。<br />
<br />
サーキットの構造上、グランドスタンドはオーバルコースのホームストレートに隣接して建てられている。ロードコースでの<a href="http://localhost/keyword/%E3%83%AC%E3%83%BC%E3%82%B9">レース</a>の場合はオーバルコースを挟んで<a href="http://localhost/keyword/%E3%83%AC%E3%83%BC%E3%82%B9">レース</a>を観戦する形になるためスタンドからコースまでの距離が遠く、来場者からは「<a href="http://localhost/keyword/%E3%83%AC%E3%83%BC%E3%82%B9">レース</a>の臨場感に欠ける」といった不満の声が根強い。このため最近では、MotoGPの日本グランプリなどの大規模イベントの場合に限り、オーバルコースのホームストレート上に仮設スタンドを組んだり（通称「1コーナー席」）、オーバルコースの一部を「アリーナエリア」として一部の観客に開放したりしている。20<a href="http://localhost/keyword/14%E5%B9%B4">14年</a>のMotoGP日本グランプリとSUPER GT最終戦ではオーバルコースのピットレーン部分に「ビクトリースタンド」と呼称した仮設スタンドを設け、より間近で観戦できるようにしたほか、オーバルコース上にはホスピタリティエリアを設けている[http://as-web.jp/news/info.php?c_id=2&amp;no=60756 もてぎビクトリースタンドは必見！ SGTなら迫力増!?],オートスポーツ,20<a href="http://localhost/keyword/14%E5%B9%B4">14年</a>10月15日。<br />

== 宿泊施設 ==<br />
場内にホテルツインリンクがある。ただし<a href="http://localhost/keyword/%E3%83%AC%E3%83%BC%E3%82%B9">レース</a>日の宿泊は関係者の宿泊が多いので要注意。<br />
茂木町・真岡市・益子町にもホテル・民宿はあるが数が少ないのでビッグ<a href="http://localhost/keyword/%E3%83%AC%E3%83%BC%E3%82%B9">レース</a>(INDYJAPANやmotoGP)開催時は栃木県と茨城県の県庁所在地である宇都宮市、水戸市のホテルも一般客の宿泊者が多くなる。<br />
<br />
== 東日本大震災の影響 ==<br />
20<a href="http://localhost/keyword/11%E5%B9%B4">11年</a>3月11日に発生した東北地方太平洋沖地震の影響を受け1か月ほど営業休止となった。4月16日に一部施設を除き営業を再開。しかしロードコースや観戦席が破損した為、コースの再舗装や縁石の入れ替えなどの改修工事が施す必要があり、MotoGP（ロード<a href="http://localhost/keyword/%E3%83%AC%E3%83%BC%E3%82%B9">レース</a>世界選手権）やスーパー耐久などの<a href="http://localhost/keyword/%E3%83%AC%E3%83%BC%E3%82%B9">レース</a>イベント開催日程を延期した。<br />
<br />
6月15日に改修工事が完了し、7月2日からの全日本ロード<a href="http://localhost/keyword/%E3%83%AC%E3%83%BC%E3%82%B9">レース</a>選手権にてイベントも含めた震災前と同様の営業再開となるが、同年のインディジャパン300については「オーバルコースの地盤に沈下・隆起が発生しており、コースの安全性や改修費用の問題がある」ことを理由にオーバルでの開催を断念し、ロードコースでの開催に変更された[http://www.twinring.jp/indyjapan/pdf/0603_indyjapan.pdf インディジャパンの開催コース変更について] - モビリティランド・20<a href="http://localhost/keyword/11%E5%B9%B4">11年</a>6月3日。この際、ロードコースのピットはチームごとに1つ割り当てられるヨーロッパ式で、インディカーでは使用できないため、オーバルのピットをロードコースと連結して使用することになった。<br />
<br />
ただ福島第一原子力発電所事故による放射性物質の飛散状況を理由に、同年8月に行われたトライアル世界選手権の日本グランプリでは、アルベルト・カベスタニーなど一部のライダーが来日を拒否し欠場した。また同年10月に行われたMotoGPの日本グランプリでも、バレンティーノ・ロッシやホルヘ・ロレンソなどの有力ライダーが一時同様の理由で<a href="http://localhost/keyword/%E3%83%AC%E3%83%BC%E3%82%B9">レース</a>をボイコットする意向を表明した[http://www.nikkansports.com/sports/motor/news/p-sp-tp2-20<a href="http://localhost/keyword/110">110</a>605-785934.html ロッシ10月日本GP来日ボイコットか] - 日刊スポーツ・20<a href="http://localhost/keyword/11%E5%B9%B4">11年</a>6月5日。ただし最終的にボイコットの意向は撤回された。<br />
