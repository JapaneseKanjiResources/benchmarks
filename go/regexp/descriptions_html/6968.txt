description<br />
&#39;<a href="http://localhost/keyword/%E5%8D%83%E8%91%89%E7%9C%8C%E9%81%93%E8%B7%AF%E6%84%9B%E7%A7%B0%E5%90%8D">千葉県道路愛称名</a>&#39;（ちばけんどうろあいしょうめい）とは、千葉県が県内大プロジェクト拠点あるいは<a href="http://localhost/keyword/%E8%A6%B3%E5%85%89%E5%9C%B0">観光地</a>と連結する広域的幹線対象の「道路愛称づくり運動」を行うこととし、1<a href="http://localhost/keyword/987%E5%B9%B4">987年</a>（昭和<a href="http://localhost/keyword/62%E5%B9%B4">62年</a>）度および、19<a href="http://localhost/keyword/88%E5%B9%B4">88年</a>（昭和<a href="http://localhost/keyword/63%E5%B9%B4">63年</a>）に県内有識者により設置した&#39;千葉県道路愛称制定委員会&#39;により、24の道路に愛称を制定したもの。<br />
<br />
愛称名を制定された道路は1<a href="http://localhost/keyword/987%E5%B9%B4">987年</a>より正式に千葉県の道路愛称名として使用されているが、それ以前からも街道として幅広く呼ばれていたものもある。しかし、東金街道は、古くは船橋から東金に通ずる御成街道が東金街道と称されていたが、現在は国道126号の東金-千葉であるなど、この制定によって古くから呼ばれていたものとは異なる愛称も存在する。<br />
<br />
== 1<a href="http://localhost/keyword/987%E5%B9%B4">987年</a>度制定 ==<br />
; 水戸街道<br />
: 国道6号の松戸市上矢切新葛飾橋-我孫子市北新田大利根橋までの区間の愛称名である。<br />
; 木下街道<br />
: 千葉県道59号市川印西線の愛称名。<br />
; 成田街道<br />
: 国道296号の船橋市前原西の中野木交差点-国道51号の成田市寺台の寺台インター。<br />
; 千葉街道<br />
: 国道14号の、市川市市川3丁目市川橋-千葉市中央区中央2丁目広小路交差点。<br />
; 佐倉街道<br />
: 国道51号の千葉市中央区中央2丁目の広小路交差点-千葉県道65号佐倉印西線の佐倉市海隣寺町の海隣寺交差点。<br />
; 東金街道<br />
: 国道126号の、東金市台方の国道128号交差点-千葉市中央区中央2丁目の広小路交差点。<br />
; 大網街道<br />
: 千葉県道20号千葉大網線の愛称名。<br />
; 茂原街道<br />
: 千葉県道14号千葉茂原線の愛称名。<br />
; 長狭街道<br />
: <a href="http://localhost/keyword/%E5%8D%83%E8%91%89%E7%9C%8C%E9%81%9334%E5%8F%B7%E9%B4%A8%E5%B7%9D%E4%BF%9D%E7%94%B0%E7%B7%9A">千葉県道34号鴨川保田線</a>の愛称名。<br />
; 房総フラワーライン<br />
: 県道南安房公園線と<a href="http://localhost/keyword/%E5%9B%BD%E9%81%93410%E5%8F%B7">国道410号</a>の一部、市道、県道千倉港線および県道和田丸山線である。<br />
