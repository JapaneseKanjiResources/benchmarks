'三木ジャンクション'（みきジャンクション）は、兵庫県三木市にある、山陽自動車道（以下山陽道と記す）のジャンクションである。

== 概要 ==
山陽道本線と山陽道木見支線・神戸淡路鳴門自動車道方面へのジャンクション。ここから神戸西ICまでは山陽自動車道木見支線となっているため、案内標識では'「明石大橋方面」'と案内されている。

== 道路 ==
* 山陽自動車道（本線）（2番）
* 山陽自動車道（木見支線）（神戸淡路鳴門自動車道連絡）（2番）

== 隣 ==
; 山陽自動車道（本線）
: (1) 神戸北IC - 淡河PA - '(2) 三木JCT' - (3) 三木東IC
; 山陽自動車道（木見支線）（神戸淡路鳴門自動車道連絡）
: '(2) 三木JCT' - (1)神戸西IC

== 歴史 ==
* 1998年（平成10年）4月5日 - 山陽自動車道 三木JCT - 神戸西ICの開通に合わせて供用開始。
