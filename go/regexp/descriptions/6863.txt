'埼玉県立庄和高等学校'（さいたまけんりつ しょうわこうとうがっこう）は、埼玉県春日部市金崎にある公立高等学校。全日制普通科の男女共学校である。

== 沿革 ==
*1980年4月 - 男女共学校として設立

== 最寄駅 ==
*東武鉄道野田線南桜井駅

== 校訓 ==
「夢　飛揚」

== 著名な卒業生 ==
*そめやゆきこ（タレント）
*東海林彬（元サッカー選手）
*菅沼実（プロサッカー選手）
