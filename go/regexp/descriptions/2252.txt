'川内インターチェンジ'（かわうちインターチェンジ）は愛媛県東温市南方にある松山自動車道のインターチェンジである。

== 概要 ==
東温市にあるインターチェンジであり、国道11号バイパスに接続しており、国道11号を西に進めば松山市街へ行くことも可能である。

=== 松山ICとの利用者分散策 ===
ETC搭載車の休日上限料金を1000円とする特別割引（いわゆる「1000円高速」）が実施された際には松山市内から高松方面へ向かう利用者に川内ICの利用を促す垂れ幕が松山環状線などの歩道橋に張られていた。また、小坂交差点（国道11号）の高架にも両ICの案内がある。

== 道路 ==
* 松山自動車道（12番）
* 接続する道路：国道11号（国道494号重複）

== 川内ICバスストップ ==
ICの外にあるため、いったん高速道路を出ることになる。なお、松山自動車道の伊予インターチェンジ延伸前は、各高速バスが「川上農協前」に発着していた。
* オレンジライナーえひめ号：新宿高速バスターミナル行（伊予鉄道・西東京バス）
* ドリーム松山号：新宿駅・東京駅・東京ディズニーランド行（JR四国バス・JRバス関東）
* オリーブ松山号：名古屋駅行（JR四国バス・JR東海バス）
* オレンジライナーえひめ号：名古屋（名鉄バスセンター）行（伊予鉄道・名鉄バス）
* 道後エクスプレスふくおか：福岡行（伊予鉄道・伊予鉄南予バス）
* 京都エクスプレス：京都駅行（伊予鉄道・京阪バス）
* オレンジライナーえひめ号：阪急梅田行（伊予鉄道・阪急バス）
* 松山エクスプレス号：神戸三宮・大阪駅行・京都駅行（JR四国バス・西日本JRバス）
* ハーバーライナー：神戸三ノ宮行（伊予鉄道・神姫バス）
* マドンナエクスプレス：岡山駅行（伊予鉄道・JR四国バス・下津井電鉄・両備バス）
* 坊っちゃんエクスプレス：高松駅行（伊予鉄道・JR四国バス・四国高速バス）
* なんごくエクスプレス：高知駅行（JR四国バス）
* ホエールエクスプレス：高知駅行（伊予鉄道・とさでん交通）
* 吉野川エクスプレス：徳島駅行（伊予鉄道・JR四国バス・徳島バス）

: なお当バス停から松山・八幡浜方面行には乗車できない。

郊外バス（一般路線バス）の最寄り停留所は、川内線および川内管内各線の「川上学校前」である。川内ICバスストップから北へ徒歩5分程度。

== 周辺 ==
* 伊予鉄道横河原線 横河原駅（徒歩約30分）
* 白糸の滝
* 川内温泉さくらの湯
* 見奈良天然温泉利楽
* 愛媛大学医学部および附属病院
