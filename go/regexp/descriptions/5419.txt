'車駅'（しゃてい、チャーチェンえき）は、台湾南投県水里郷にある台湾鉄路管理局集集線の駅。同線の終着駅である。

== 駅構造 ==
*島式ホーム1面1線の地上駅。

== 利用状況 ==



== 駅周辺 ==
*明潭ダム
*車酒庄
*振昌木業(林班道商圈 The Grove,Taiwan)
*水里郷車埕村村落
*南投県水里郷車埕国民小学校

== 歴史 ==
*1922年1月14日 -  '外車駅'として開業。
*1961年 - '車駅'に改称。
*1999年9月21日 - 921大地震により被害を受ける。
*2001年7月21日 - 駅の再建が完了。

== その他 ==
*「(てい)」の字は台湾語の方言字で「場」の意味。昔ここは埔里行き人車軌道の手押車が多かったが現在は廃止されている。
*通常、日本の常用漢字にはないため「土呈」「呈」「程」と称す場合が多い。
