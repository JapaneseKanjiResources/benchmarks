'チェック'

== Check ==

* 検査
* チェックシートやコンピュータのチェックボックス等において選択・照合に使う「&#10003;」「&#10007;」などの記号。チェックマーク。また、この記号を付けること。
* 模様の一種で、格子模様と呼ばれる物の総称。アーガイル柄、市松模様、ウインドウペーン、ギンガム、グラフ（方眼紙）チェック、グレンチェック、タッタソールチェック、タータンチェック、千鳥格子（en）、バーバリーチェック等。
* チェス用語。将棋の王手にあたる。王手#チェスの王手（チェック）を参照。
* サッカーのプレーを表す用語の一つ。ディフェンス (サッカー)を参照。
* アルティメット競技において、反則などでプレー停止後、再開前に選手のポジションなどを確認した後、再開すること。
* C言語向けオープンソースのユニットテストフレームワーク。Check (ユニットテスト・フレームワーク)を参照。

== Cheque ==
、
* 小切手。日本ではトラベラーズチェックなどの単語に用いられる。

== Cech ==
* トーマス・チェック (Thomas Robert Cech) - アメリカ合衆国の分子生物学者、生化学者。

== Czech ==
'チェック' (:en:Czech) は、チェコ語のČechy（チェヒ）に由来する英語表記で、元来はボヘミアを指すチェコ関連の語。日本では伝統的に'チェック人'（チェコ人）の語に用いられている。なお、英語で単に''Czech''と言うとチェコ語を指し、国家としてのチェコ共和国は''Czech Republic''、チェコ人は''Czechs''と言い表される。
