改札口（2008年11月）
'伊豆北川駅'（いずほっかわえき）は、静岡県賀茂郡東伊豆町奈良本1225-12にある、伊豆急行伊豆急行線の駅。

== 駅構造 ==
単式ホーム1面1線を有する地上駅。

; のりば


=== 構内 ===
* SuicaおよびSuicaと相互利用可能なICカード（PASMO等）は、伊豆急行線内全駅で利用可能であるが当駅にチャージ機は設置されずチャージサービスには対応していない。

== 利用状況 ==
* 2007年度の1日平均乗降人員は159人（静岡県統計年鑑による）

== 駅周辺 ==
* 北川温泉
* アクアポート伊豆
* 穴切湾
* 国道135号

== 路線バス ==
* 東伊豆町自主運行バス（南伊豆東海バス）「北川」停留所　駅から徒歩約10分（国道135号線 伊豆急行線との交差下付近）

== 歴史 ==
* 1964年（昭和39年）10月1日 - 開業。
* 2012年（平成24年）4月1日 - 終日無人駅化。
