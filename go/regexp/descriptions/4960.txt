'伊比井駅'（いびいえき）は、宮崎県日南市大字伊比井にある、九州旅客鉄道（JR九州）日南線の駅である。

== 駅構造 ==
島式ホーム1面2線を有する地上駅。駅舎とホームは高い場所にあるため、道路からは階段を渡って行くこととなる。小ぶりながら立派なコンクリートの駅舎が残る無人駅である。

== 利用状況 ==
* 2012年度の1日平均乗車人員は23人である。


== 駅周辺 ==
* 鵜戸神宮
* 冨土海水浴場

== 歴史 == 
* 1963年（昭和38年）5月8日 - 開業。
* 1987年（昭和62年）4月1日 - 国鉄分割民営化により九州旅客鉄道に承継。

== その他 == 
* サントリー飲料水「緑水」TVCM（2002年）のロケ地として使われる（出演：宮崎あおい）。
* 小池里奈DVD「里奈は本日天然なり」で一部ロケ地として使われる。
* 資生堂化粧水「エリクシールホワイト」のTVCM（2010年）のロケ地として使われる（出演：広末涼子）。
