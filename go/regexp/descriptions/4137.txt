ホーム（駅舎脇から、2008年11月）
'稲梓駅'（いなずさえき）は、静岡県下田市落合92-1にある伊豆急行伊豆急行線の駅。普通列車のみが停車する。

== 駅構造 ==
島式ホーム1面2線を有する地上駅。

; のりば

※通過列車待避の場合など、のりばが変更になる列車がある。

=== 構内 ===
* SuicaおよびSuicaと相互利用可能なICカード（PASMO等）は、伊豆急行線内全駅で利用可能であるが当駅にチャージ機は設置されずチャージサービスには対応していない。

== 利用状況 ==
* 2007年度の1日平均乗降人員は131人（静岡県統計年鑑による）

== 駅周辺 ==
山に囲まれている。周辺に観光地はなく、農村地帯に民家が見られるのみである。

* 下田市立稲梓小学校
* 下田市立稲梓中学校
* 米山薬師
* 下田達磨大師
* [http://www.uehara-buddhismart.or.jp/index.html 上原仏教美術館]
* [http://www.uehara-museum.or.jp/ 上原近代美術館]
* 国道414号

== 路線バス ==
* 西伊豆東海バス 落合停留所 - 駅から徒歩約10分、松崎・堂ヶ島方面行のバスが発着。

== 歴史 ==
* 1961年（昭和36年）12月10日 - 開業。
* 2012年（平成24年）4月1日 - 終日無人化。

== 駅名の由来 ==
駅の設置場所にちなむ。開業当時は賀茂郡下田町だったが、この地が下田町になる前に稲梓村の中心地だった事が由来。
