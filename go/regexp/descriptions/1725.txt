ホーム（2008年9月）
'天王町駅'（てんのうちょうえき）は、神奈川県横浜市保土ケ谷区天王町字2丁目にある、相模鉄道本線の駅である。駅番号は'SO04'。電報略号はテン。

== 駅構造 ==
相対式ホーム2面2線を有する高架駅。改札口は横浜方に1か所設置されている。改札内コンコースと各ホームを連絡するエレベーターが設置されている。2016年（平成28年）7月16日からは下りホームでエスカレーターが使用開始された。

駅前広場は旧帷子橋跡である。

当初、当駅 - 星川駅間の連続立体交差事業により、島式ホーム1面2線に変更される予定であった[http://www.sotetsu.co.jp/train/crossover/gaiyo.html 相鉄グループHP「連続立体交差事業」]が、計画変更により工事完成後も相対式ホーム2面2線のままとなる[http://www.sotetsu.co.jp/train/crossover/situation_024.html 相鉄グループHP「連続立体交差事業」現在の工事状況]なお駅構内図についてはhttp://www.sotetsu.co.jp/train/station/pdf/tennocho_01.pdfより。。

== 駅設備 ==
改札を入って奥まで進むと男女別にトイレがある、コンコース内にはそば屋があり、隣接して売店も設けられている。

== 利用状況 ==
1日あたり乗車人員の推移
2014年度の1日平均乗降人員は'26,835人'であり[http://www.sotetsu.co.jp/csr/data/pdf/joko2014.pdf 相鉄グループHP「相鉄乗降人員」]、相鉄本線内の優等列車通過駅の中では最も多い。しかし、近年は減少傾向にある。

=== バス路線 ===
駅東側にある旧東海道上にある'天王町駅前'が最寄り停留所となる。以下の路線が横浜市交通局（市営）、神奈川中央交通（神奈中）、相鉄バス（相鉄）により運行されている。
==== 保土ヶ谷駅西口方面のりば ====
* 25 星川駅前・花見台経由保土ヶ谷駅西口行（市営）
* 121 大門通経由保土ヶ谷駅西口行（神奈中）
* 浜16 和田町駅前・市沢町経由西原団地行（相鉄）

==== 国道16号線方面のりば ====
* 25 浅間町車庫前経由横浜駅西口行（市営）
* 深夜バス 保土ヶ谷車庫前行（市営）
* 121 上星川・羽沢団地前経由新横浜駅行（神奈中）
* 浜16 浅間町車庫前経由横浜駅西口行（相鉄）

== 歴史 ==
* 1930年（昭和5年）9月10日 - 開業。
* 1945年（昭和20年）5月29日 - 横浜大空襲により駅舎焼失。
* 1948年（昭和23年） - 駅舎再建。
* 1968年（昭和43年）3月27日 - 駅舎高架化。本線唯一の高架駅に。

== 駅名の由来 ==
鎌倉時代に創建された「牛頭'天王'社（ごずてんのうしゃ）」から。

== その他 ==
歌川広重作『東海道五十三次』の「'程ヶ谷'」は、当駅前南側を流れていた改修前の帷子川に架かる橋を描いたものである。
