'上枝駅'（ほずええき）は、岐阜県高山市下切町にある、東海旅客鉄道（JR東海）高山本線の駅である。難読駅名の一つ。駅名は開業当時存在した上枝村に由来する。

== 歴史 ==
* 1934年（昭和9年）10月25日 - 高山本線の飛騨小坂 - 坂上駅間開通と同時に開業。旅客及び貨物の取扱を開始。
* 1982年（昭和57年）3月1日 - 貨物の取扱を廃止。
* 1987年（昭和62年）4月1日 - 国鉄分割民営化により、東海旅客鉄道（JR東海）の駅となる。

== 駅構造 ==
相対式ホーム2面2線を持ち、列車交換が可能な地上駅。古い木造駅舎が線路より1段低いところに置かれている。2番線ホームには待合室はない。駅舎は離れたところに設置されている。駅舎の中には時刻表やベンチが設けられている他、地元の小学生が作った、機関車の模型が置いてある。高山駅管理の無人駅。

== 利用状況 ==
* 2013年度の1日平均乗車人員は19人である。


== 駅周辺 ==
周辺は住宅地である。また駅前の道を東に200メートルほど行くと国道41号に至る。

* 高山上枝駅前郵便局
* 高山警察署上枝警察官駐在所
* 高山西高校
* 温泉 ひだまりの湯
* 高山市立三枝小学校
* 須田病院
* 岐阜県道457号名張上切線
* 宮川
* 宮川第14号橋梁（宮川と国道41号線にかかる橋梁で、書籍では『あじめ橋』と紹介されていて、鉄道写真の名所）
* 高山インターチェンジ

== バス路線 ==
駅近くの岐阜県道457号上にコミュニティバスのらMyCar北線が停車する。
