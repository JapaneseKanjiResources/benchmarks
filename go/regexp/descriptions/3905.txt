'インディペンデント'（independent）とは、独立の意。

== メディア ==
* インデペンデント（The Independent） - イギリスの新聞。
* ITV (イギリス)（Independent Television）- イギリス最大の民間放送局。

== エンターテインメント ==
* インディーズ - インディペンデント・レコード・レーベルほか、映画などはこちらも参照。
* インディペンデント映画 - 自主映画の事。

== スポーツ ==
* インディペンデント団体 - プロレス界における、小規模プロレス団体の事。通称「インディ」。対義語は「メジャー団体」。　詳細はプロレスを参照。
* インディペンデント・リーグ - アメリカの野球界において、メジャー・リーグおよびマイナー・リーグ以外の独立した組織として存在する野球リーグの事。独立リーグ。

== 経済 ==
* インディペンデント - シチズンホールディングス株式会社が展開する腕時計のブランド。
