'神田橋ジャンクション'（かんだばしジャンクション）は、東京都千代田区にある首都高速道路のジャンクションである。

== 連絡している路線 ==
都心環状線（竹橋JCT方面）⇔都心環状線（江戸橋JCT方面）・八重洲線
=== 分岐標識表記 ===
（竹橋方面より）
:［↑銀座 箱崎〔C1〕〔6〕〔9〕〔7〕｜〔Y〕〔11〕羽田 丸の内／］

== 隣 ==
; 首都高速都心環状線
: (26)北の丸出口 - 竹橋JCT - (27)神田橋出入口 - '神田橋JCT' - (29)神田橋出入口 - (30)呉服橋出入口

; 首都高速八重洲線
: '神田橋JCT' - 八重洲駐車場 - (44)丸の内出口

== 注意点 ==
* 八重洲線⇔神田橋出入口の利用はできない。
* 八重洲線は大型車・危険物積載車の通行は禁止されている。
* 八重洲線と都心環状線内回りの合流は、見通しが悪く、合流車線長が短い。
