'名古屋市交通局5050形電車'（なごやしこうつうきょく5050がたでんしゃ）は、1992年（平成4年）に登場した名古屋市交通局（名古屋市営地下鉄）東山線用の通勤形電車。

== 車両概要 ==
東山線に残っていた300形などの非冷房車を置き換える目的で、1992年（平成4年）から2000年（平成12年）にかけて6両編成27本（162両）が日本車輌製造で製造された。

車体や搭載機器は名城線・名港線2000形をベースとして、前頭形状を5000形に似せてあるが、傾斜の仕方や灯具の配列などが異なる。座席はバケットタイプだがモケットの色は5000形と同様で、赤橙色の濃淡で着席区分を表す。また、荷棚のパイプも一般的なレール方向とした（2000形は枕木方向）。車両長は15.58mと短い。
5000形（左）と5050形（右）の先頭部の比較

== 車内設備の差異 ==
*1992年度導入の5151 - 5153編成と1993年度導入の5154編成以降では荷棚の端部形状が異なる。
*1994年度導入の5156編成以降は詰め物材質がポリエステル素材に、座面形状が薄型にそれぞれ変更されている。
*1995年度導入の5158編成以降は車内の扉上に路線図式車内案内表示器を3箇所千鳥配置し、ドアチャイムを設置している。

== 編成 ==
藤が丘方面先頭から4両目に当たる5450形は平日の始発から終電まで女性専用車両となる。


== ラッピング ==
2001年からはコカ・コーラなどのラッピング広告が掲出された編成もある。

== 製造後の改造など ==

=== CS-ATC化改造 ===
2004年3月に保安装置が打子式ATSからCS-ATCに切り替えられた際に、ブラックパネルが特徴の運転台に改造され、同時に速度計がデジタル式からアナログ式に交換された。

=== ATOの搭載・ワンマン運転・ホームドア対応改造 ===
本形式についても初期車の製造から既に20年が経過したため、2012年7月からATOの搭載・ワンマン運転・ホームドア対応改造が施工されることになった。2015年4月までにATO導入によるワンマン運転化・ホームドア設置に向けて全編成が制御方式のソフト変更（名城線・名港線2000形2123・2128編成以降と同様のベクトル制御）と車内自動放送の更新を完了している。
