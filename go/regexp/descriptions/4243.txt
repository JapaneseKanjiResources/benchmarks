'澤村 宗十郎'（さわむら そうじゅうろう）は、歌舞伎役者の名跡。屋号は紀伊國屋。定紋は丸にいの字、替紋は花有り足有りの笹竜胆。

代々の墓は田島山十一ヶ寺受用院にある。

*澤村宗十郎（初代澤村長十郎）
**初代の師匠・初代澤村長十郎 (1675–1734) が一時期「宗十郎」を名乗ったが、その後すぐに「長十郎」に復した。代々には数えない。
**初代澤村長十郎 → 澤村宗十郎 → 初代澤村長十郎

*初代 澤村宗十郎
**初代澤村長十郎の門弟、1685–1756。
**初代染山喜十郎 → 澤村善五郎 → 澤村惣十郎 → 初代澤村宗十郎 → 三代目澤村長十郎 → 初代助高屋高助

*澤村宗十郎（遥波宗十郎）
**初代の門弟・澤村春五郎 (?–1749) が、初代が「三代目長十郎」を襲名したことにともない「宗十郎」となったが、その翌年には「惣十郎」に改名、その後すぐに死去した。その俳名「遥波」をとって遥波宗十郎と通称される。代々には数えない。
**染川市十郎 → 澤村春五郎 → 澤村宗十郎 → 澤村惣十郎

*二代目 澤村宗十郎
**初代の養子、1713–70。はじめ初代富澤門太郎の門弟。
**富澤長之助 → 竹中歌川 → 瀧中歌川 → 歌川四郎五郎 → 二代目澤村宗十郎

*三代目 澤村宗十郎 
**二代目の次男、1753–1801。俳号から俗に「大訥子」。
**初代澤村田之助 → 三代目澤村宗十郎

*四代目 澤村宗十郎
**三代目の長男、1784–1813。
**初代澤村源之助 → 四代目澤村宗十郎

*五代目 澤村宗十郎
**四代目の門弟、1802–53。
**初代澤村源平 → 二代目澤村源之助 → 初代澤村訥升 → 五代目澤村長十郎 → 三代目助高屋高助 → 五代目澤村宗十郎

*贈六代目 澤村宗十郎
**五代目の長男、1838–86。襲名間近に名古屋で出演中に倒れる。死後六代目宗十郎を追贈。
**二代目澤村源平 → 二代目澤村訥升 → 三代目助高屋高助 → 贈六代目澤村宗十郎

*七代目 澤村宗十郎
**贈六代目の養子、1875–1949。
**四代目澤村源平 → 三代目澤村訥升 → 七代目澤村宗十郎

*八代目 澤村宗十郎
**七代目の三男、1908–75。
**五代目澤村源平 → 四代目澤村訥升 → 市川松莚 → 四代目澤村訥升 → 八代目澤村宗十郎

*九代目 澤村宗十郎
**八代目の長男、1933–2001。
**六代目澤村源平 → 五代目澤村訥升 → 九代目澤村宗十郎
