'広島県道276号矢野海田線'（ひろしまけんどう276ごう やのかいたせん）は広島市安芸区矢野新町2丁目と安芸郡海田町曽田を結ぶ一般県道である。

== 概要 ==
* 起点：広島市安芸区（矢野新町2丁目・消防署矢野出張所前交差点 = 広島県道34号矢野安浦線バイパス交点）
* 終点：安芸郡海田町（曽田・畝橋東交差点 = 国道2号交点）
* 総延長：約4.3km
* 実延長：約4.3km

== 沿革 ==
* 1990年2月13日 広島県告示第162号により認定される。

== 通過する自治体 ==
* 広島市（安芸区） - 安芸郡海田町

== 沿線の地理 ==
=== 主要施設 ===
* 海田地区消防署矢野出張所
* 陸上自衛隊海田市駐屯地（第13旅団）
* 広島県立海田高等学校
* 広島県警海田警察署
* 海田郵便局
* 広島国際学院高等学校

== 主な橋梁 ==
* 西明神橋（安芸郡海田町西明神町 - 安芸郡海田町南明神町）

== 備考 ==
* 広島市安芸区矢野新町2丁目・海田大橋入口交差点 - 安芸郡海田町曽田は国道2号東広島バイパスおよび広島南道路の側道として整備されている。特にJR呉線以東で道路の中央部に大きな空き地ができているのはそのためである（JR呉線以西も着手予定）。また、JR呉線以西でかつて自衛隊の社宅であったところも、現在では老朽化に伴う移転により、大きな空き地となっており、工事の着手が進んでいる。
