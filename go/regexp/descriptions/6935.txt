'大洲冨士インターチェンジ'（おおずとみすインターチェンジ）は、愛媛県大洲市にある松山自動車道（国道56号大洲道路）のインターチェンジである。

== 道路 ==
* 松山自動車道（国道56号大洲道路）
* 接続する道路：国道197号

== 周辺 ==
* 冨士山

== 隣 ==
; 松山自動車道
: 大洲北IC - '大洲冨士IC' - 大洲肱南IC
