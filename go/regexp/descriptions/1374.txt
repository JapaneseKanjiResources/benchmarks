== 固有名称 ==
* 国際スピード郵便 () - 各国の郵便で行われている海外への配達サービスの1つで、追跡システムを持つ。
* Expanded Memory Specification - MS-DOS上でのメモリ拡張手法。
* ヨーロッパ通貨制度 ()
* エレクトロニック・ミュージック・スタジオ () - イギリスの電子楽器メーカー。
* エディンバラ数学会 () - イギリス スコットランドの数学学会。
* ヨーロッパ数学会 () - ヨーロッパの学術団体。
* 拡張メッセージサービス () - ショートメッセージサービスを拡張したサービス。
* ヨーロッパ震度階級 () - ヨーロッパ諸国などで使われている震度階級。
* E・M・S・ナンブーディリパド () - インドの政治家。
* エコドライブ管理システム () - トラックのデジタルタコメーターを用いた省エネルギーシステム。

== 一般名称 ==
* 電子機器の受託生産 () - 製品製造および開発工程のアウトソーシング。
* 環境マネジメントシステム () - 企業などの活動を、環境にいかに負担をかけないものにするかを追求するシステム。
* アルキル化剤 ()
* 救急隊 () - 消防本部の救急隊員が身につけている衣類や救急車などにマーキングされていることがある。各々の救急隊員はEMT ()。
* 電磁スペクトル ()
* 電気的筋肉刺激 () - 低周波#医療での低周波を参照。低周波以外も使われる。
* エネルギー管理システム () - 電力供給や省エネのために電力の運用を管理するシステム。
* 好酸球増加筋肉痛症候群 ()
* 早期死亡症候群 () - エビの病気。
* エチルメタンスルフォン酸 ()
* 電磁サスペンション () - 電磁力で浮上することによるサスペンション。
* 電子マニフェストシステム () - 積荷目録（マニフェスト）を電子化したシステム。貿易などで使われる。日本で産廃処理追跡に使われているマニフェスト制度もその1つ。
