description
'オレンジ' (orange ) は、柑橘類の植物およびその果実。また、熟れたオレンジの果実の外皮の色は'オレンジ色'と呼ばれる。

== 地名 ==
* 'オレンジ郡' - アメリカ合衆国の郡。複数ある。
* オランジュ - フランスの都市（フランス語：Orange、英語読みでオレンジ）。
* オレンジ自由国 - かつてアフリカに存在した国家。
* オレンジ (カリフォルニア州) - アメリカ合衆国の都市。

== 人物・グループ ==
* オレンジ (アニメーター) - 日本のアニメーター。別名、ORANGE。
* オレンジ (お笑い) - よしもとクリエイティブ・エージェンシー東海支社に所属しているお笑いコンビ。
* Orange Lounge - BEMANIシリーズのTOMOSUKE（舟木智介）と徳井志津江のユニット。
* ORANGE RANGE - 日本のミクスチャー・ロックバンド。
* G.I.オレンジ - イギリスのニュー・ロマンティックバンド。
* R-ORANGE - 日本の音楽アーティスト。

== 企業・組織・企業ブランド ==
* オレンジ (楽器メーカー) - イギリスのギターアンプメーカー。
* オレンジ (アニメ制作会社) - 日本のアニメ制作会社。
* 100円ショップ オレンジ - 関西・中部地方に出店している100円ショップチェーン。
* 肥薩おれんじ鉄道 - 鹿児島県の鉄道会社。
* オレンジ共済組合 - オレンジ共済組合事件を起こした日本の組織。
* Orange (通信会社) - フランスの電気通信事業者。旧称：フランステレコム。従来は同社傘下の携帯電話キャリアのブランドとして使用されてきたが、2006年から固定電話なども含むグループ全体の統一ブランドとなり、2013年から社名に採用された。
