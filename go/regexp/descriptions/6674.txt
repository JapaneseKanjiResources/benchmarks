'出雲警察署'（いずもけいさつしょ）は、島根県警察が管轄する警察署の一つである。

== 所在地 ==
*島根県出雲市塩冶有原町2丁目19番

== 管轄区域 ==
*島根県出雲市

== 沿革 ==
*2005年4月1日 （旧）出雲警察署・平田警察署・大社警察署を統合し、（新）出雲警察署となる。

== 広域交番 ==
（）の中は所在地。
*平田広域交番（出雲市平田町）
*大社広域交番（出雲市大社町杵築東）
