'埼玉県立狭山工業高等学校'（さいたまけんりつ さやまこうぎょうこうとうがっこう）は、埼玉県狭山市富士見二丁目にある公立工業高等学校。

== 沿革 ==
* 1962年4月 - 開校式。機械科（定員360名）
* 1963年4月 - 電気科（定員360名）増設
* 1988年4月 - 電子機械科（男女共学・定員80名）設置

== 主な行事 ==
* 6月 - 体育祭
* 7月 - 球技大会　夏
* 9月 - 水泳大会
* 10月中旬 - 芸術鑑賞教室
* 10月下旬 - 狭工祭
* 11月 - マラソン大会
* 12月 - 球技大会　冬

== 交通 ==
西武新宿線 狭山市駅東口下車、徒歩12分。

== 卒業生 ==
* 増岡浩（パリ・ダカール・ラリー優勝ドライバー）

== 制服 ==
学生服で、上下とも黒。校章は左首に、右首には、学年章である。赤、青、緑に分けられている。
学年章には、学科も分かるようになっており、「M」が機械科、「E」が電気科、「S」が電子機械科。

== その他 ==
*ドラマ『理想の息子』（2012年1月放送。日本テレビ）ロケ地。
*映画『カノジョは嘘を愛しすぎてる』（2013年12月公開。東宝）ロケ地。
