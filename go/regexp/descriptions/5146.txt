'美川温泉'（みかわおんせん）は、石川県白山市（旧国加賀国）にある温泉。

== 泉質 ==
* 含重曹食塩泉
* 弱食塩泉
**源泉温度38～48℃
**毎分湧出量285リットル
**黒色をした源泉

== 温泉街 ==
手取川の河口近くに3軒の旅館、ホテルが存在する。

共同浴場はないが、温泉地にある旅館にて日帰り入浴が可能である。

== 歴史 ==
開湯は1971年である。

== アクセス ==
* 鉄道：北陸本線美川駅より徒歩約10分。
* 車：北陸自動車道美川ICより約5分。
