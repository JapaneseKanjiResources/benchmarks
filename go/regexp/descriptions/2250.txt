'トロッコ亀岡駅'（トロッコかめおかえき）は、京都府亀岡市篠町山本にある嵯峨野観光鉄道嵯峨野観光線の駅。電報略号は「トロカオ」。

「亀岡駅」と名乗ってはいるが、西日本旅客鉄道（JR西日本）山陰本線の亀岡駅からは東へ約3キロメートル離れており、同線馬堀駅の方が近い。

== 歴史 ==
* 1991年（平成3年）4月27日 - 開業。
* 1992年（平成4年）4月20日 - 駅舎が完成する。

== 駅構造 ==
単式ホーム1面1線の地上駅。築堤上にあるため、道路から駅舎へは階段を昇る必要があったが、2009年1月からの運休期間中に道路から駅舎を経由せず直接ホームに至る、車椅子等専用のスロープが設置された。

== 駅周辺 ==
* JR西日本山陰本線（嵯峨野線）馬堀駅 - 本駅から線路沿いに西方徒歩10分（500メートル）。
* 亀岡市立病院
* 医療法人社団瀬尾医院
* 亀岡市立安詳小学校
* 鵜川

== 路線バス ==
