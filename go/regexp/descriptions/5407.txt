'北真岡駅'（きたもおかえき）は栃木県真岡市にある真岡鐵道真岡線の駅である。

== 歴史 ==
* 1955年（昭和30年）4月1日 - 日本国有鉄道の駅として開業。
* 1987年（昭和62年）4月1日 - 国鉄分割民営化により東日本旅客鉄道（JR東日本）に継承。
* 1988年（昭和63年）4月11日 - 真岡鐵道に転換。同時に呼称を「きたもうか」から「きたもおか」に変更。

== 駅構造 ==
単式ホーム1面1線を有する地上駅で無人駅。駅舎はないが、車両約1両分の屋根付き待合所が設置されている。一般道からは駅南側より出入りする。ホームへ昇降する階段が2か所設置されているが、車椅子対応のスロープはない。

駅前には駐輪場、送迎一時利用用駐車場、公衆電話、公衆便所がある。

== 利用状況 ==
2012年度の一日平均乗車人員は104人である。

== 駅周辺 ==
* 栃木県道156号石末真岡線 - 駅の脇（真岡方）を横断している。
* 栃木県健康福祉センター - 駅南方向へ。
* サンキ真岡店　- 駅南方向へ。
* 萩田稲荷神社
* 高根沢内科クリニック（個人開業のクリニック） - 駅南方向へ。
* カワチ真岡東店 - 駅南方向へ。
* 真岡市立真岡小学校
* 真岡観光リス村 - 駅東方徒歩10分ほど。線路脇なので列車から見ることができる。リス、プレーリードッグ、リスザル、ウサギ、ミニチュアホースなどが飼育、公開されている。
* 大前恵比寿神社

== バス路線 ==
最寄バス停は駅南側・県道を約150mの「保健所前」バス停、または駅北側・県道を約400mの「中郷」バス停となる。


== その他 ==
