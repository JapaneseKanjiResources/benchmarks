'向瀬上駅'（むかいせのうええき）は、福島県福島市瀬上町にある阿武隈急行線の駅。

キャッチフレーズは、「ももの里」。

== 駅構造 ==
単式ホーム1面1線の地上駅である。駅舎はなく、ホームに待合室があるだけの無人駅となっている。

== 利用状況 ==
* 阿武隈急行 - 2014年度の1日平均乗車人員は、27人である[https://www.pref.fukushima.lg.jp/uploaded/attachment/161504.xls 第130回福島県統計年鑑]。
* 近年の乗車人員は以下の通りである（いずれも1日平均）。



== 駅周辺 ==
周囲には果樹園が多い。伊達市と近接しており、そちらからの利用者も多い。

2001年まで、駅東側の沼西地区に高子沼グリーンランドが存在し、最寄り駅となっていたが現在は閉園・撤去されている。

* 阿武隈川

== 歴史 ==
* 1988年（昭和63年）7月1日 - 開業。

== その他 ==
* 当駅は2006年にTBSテレビで放映されたドラマ『花嫁は厄年ッ!』で、“桃山駅”としてロケが行われた。
